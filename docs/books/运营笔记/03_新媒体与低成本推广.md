# 第五部分 新媒体运营 + 第六部分 低成本推广

## 🎯 核心主张

**新媒体只是「品牌的宣传阵地」，不是你的用户**——微博/微信/视频号上的粉丝本质是平台的用户，你拿不到他们的数据。所以做新媒体的真正目标是把这些"借来的用户"沉淀到自己可控的资产里（社群、付费体系、CRM）。

**做新媒体的最高境界叫「人神合一」**：你的气质、思维、价值观跟账号完全融合成一个人格。

---

## 📌 关键框架

### 框架一：新媒体的6条底层认知（直接照抄）

| # | 认知 | 操作含义 |
|---|------|---------|
| 1 | 招对运营人 | 账号气质=运营人气质，搞笑号找不到木讷的人 |
| 2 | 花钱但不堆钱 | 30+账号 = 6个人运营900万粉丝（人少不是问题） |
| 3 | 质量比数量重要 | 1000万粉丝的招商银行不如几万粉丝的冷兔活跃 |
| 4 | 新媒体是宣传阵地，不是你的用户 | 微博/微信上的粉丝属于平台 |
| 5 | 玩法比你想的多 | 微博除了发文还有活动、投票、话题排行榜 |
| 6 | 用户转化难度大 | 1000万粉丝转到APP/社区<1% |

### 框架二：新媒体的5个基本特性

1. **个性**——释放运营人个性，引起共鸣
2. **口碑**——不是热点都跟，要做正面正能量内容
3. **传播性**——做内容前先问"用户为什么帮你传播？"
4. **平台选择**——不是所有平台都做，做不起来要会放弃
5. **灵活性**——1年前的玩法可能就废了，要持续试错

### 框架三：「人神合一」的3个把关点

| 环节 | 把关重点 |
|------|---------|
| 招聘 | 气质（先天的） + 可塑性 + 稳定性（换人=账号气绝） |
| 定位 | 不能为粉丝数偏离定位 |
| 监督 | 目标设定→监督→执行→考核→奖惩五个环节标准化 |

### 框架四：四大平台运营要点速查

| 平台 | 玩法重点 |
|------|---------|
| **微博** | 微博活动+找真实大号互转+跟热点 |
| **微信** | 强调原创+传播性+转发率比粉丝数重要 |
| **贴吧** | 找靠谱吧主 + 接触官方花钱开权限 |
| **QQ空间** | 依赖官方资源 + 日志保持频次（3-4条/天） + "说说"要凑够140字 |

### 框架五：网红炒作的5步流程（即"事件营销"的本质）

```
发现热点 → 策划 → 执行 → 轰炸/跟踪 → 揭秘/曝光
```

**网红能持久走红的3个特征**：
1. 用最新平台（2010年前论坛、2010年后微博、再后是自媒体）
2. 持续轰炸——不是一个点，要多个点循环曝光
3. 能贴上专属标签——「春哥」、「奶茶妹妹」、「叫兽易小星」

**关键洞察**：标签不一定是文字，可能是画面——提到天仙MM就是那身行头，提到奶茶MM就是手捧奶茶的清纯形象。

### 框架六：App推广6条+PC推广3条

**App推广**：
1. 应用商店（首发+ASO的9要素）
2. 社会化营销（热点+口碑）
3. 互联网开放平台
4. 预装（每台8-10元成本）
5. 论坛/贴吧推广（软文）
6. 新闻投稿（虎嗅、36氪、i黑马）

**ASO的9个因子**：应用名称、关键字、描述、icon、截图视频、评价、安装量、活跃用户量、社会化分享数据。

**PC推广**：
1. 资源互换（推荐位/广告位互换，按数据比对：Alex排名+人均PV）
2. 提升网站PR值（友情链接质量比数量重要，50个高质量足矣）
3. 检测友情链接效果（必须"互链"——通过对方页面能跳回自己）

### 框架七：微博/微信推广3要素

| 要素 | 重点 |
|------|------|
| 内容运营 | 必须原创 + 站在用户角度问"我会不会转发" |
| 用户运营 | 微信公众号配微信群（基于功能做用户体系） |
| 事件营销 | H5页面是不错的方式（虎扑NBA名字替换案例：几天400万分享） |

## 🔧 可落地的交付物

### 交付物1：账号定位自检表（懂叔账号专用）

每次发内容前对照：
- [ ] 这条内容跟"出海+新加坡商业洞察"定位完全契合吗？
- [ ] 我会不会主动转发这种内容？
- [ ] 这条内容能用几个关键词概括"懂叔气质"？
- [ ] 我是不是在为了涨粉而做不相关的内容？
- [ ] 这条内容会让"对的粉丝"留下来吗？

**类延昊原话**：「不是所有用户都有价值，只有目标用户才最有价值。」

### 交付物2："人神合一"检查清单——内容平台招人/找搭子用

招做账号助手时对照：
- [ ] 这个人骨子里有"出海创业人"的气质吗？（不是装出来的）
- [ ] 他读懒得读的长文能耐住性子吗？（执行力）
- [ ] 他能保持长期稳定吗？（换人 = 账号气绝）
- [ ] 他对"懂叔"账号气质有共鸣还是只把它当工作？

### 交付物3：H5/视觉化内容传播框架（套用虎扑NBA案例）

虎扑做火的逻辑：**让用户把自己代入名人位置 + 分享得到满足感**。

**关键**：低成本但产生「分享的成就感」。

### 交付物4：内容分享前自检三问

每次发文/口播前过一遍：
1. **传播性**：用户分享这条之后能得到什么？（情绪/身份/利益/谈资）
2. **原创度**：这是不是Ctrl+C+V的内容？
3. **是否自嗨**：写完读一遍，问"我自己会转发吗？"

### 交付物5：搜索/SEO的5个落地行动（如适用）

| 平台 | 直接动作 |
|------|---------|
| 视频号 | 标题前置关键词，简介带3-5个核心词 |
| 公众号 | 文章标题15-32字，关键词钩子≥4个 |
| 小红书/抖音 | 善用平台搜索热词（不是凭感觉编标题） |
| 个人品牌 | 长期沉淀一两个专属标签（如"新加坡懂叔"，比"懂叔"更精准） |

### 交付物6：网红/IP人设的6条禁忌

打造个人IP时绝对不能踩：
- [ ] 不碰低俗/暴力/政治红线
- [ ] 不要透支自己（适可而止，永远留后路）
- [ ] 不要因为红了就不可一世（捧得越高摔得越惨）
- [ ] 不要变成"水军式"运营（目的赤裸裸只为钱）
- [ ] 不要用过时的标签（"网游第一美女"满大街了就废了）
- [ ] 不要轻易换运营人（账号气质会断）

## 💡 重要洞见与案例

### 洞见1：「自有平台才是你的用户」

> 凡是你拿不到用户资料的，比如微博、微信，那是它们的用户，不是你的用户。

这条直接戳破"我有100万粉丝"的幻觉。真正可控的资产：
- 微信社群（你能看到每个人）
- 自己的CRM/付费用户名单
- 自己的小程序/APP

**给懂叔的启示**：公众号+视频号的粉丝是"借来的"，要持续往自己的群、付费星球、私域客户里沉淀。

### 洞见2：人少胜过堆人头

作者公司：6个人运营900万粉丝、30+账号、光贴吧10个。

「不是向堆人头的角度发展，而是更倾向于每个人所贡献的价值。」

**这对个人IP创业者特别有启发**：与其招10个抄写工，不如自己练成一个"人神合一"的运营。

### 洞见3：「奶茶MM案例」——做长红需要轮番轰炸

奶茶MM能持续走红多年的原因：
- 2009年：手捧奶茶的照片（起点）
- 上清华（背景升级）
- 绯闻男友（话题）
- 奶东恋（事件）
- 微软"小冰"实习（专业身份）
- 大学毕业（成长线）
- 澳大利亚结婚（爱情线）
- 怀孕（家庭线）

**洞察**：长红的人/品牌都是有"叙事弧线"的，不是单点爆款。

### 洞见4：虎扑NBA H5传播案例

虎扑做了一组NBA名场面文章，允许用户**把主角名字换成自己的名字**分享出去。几天400万分享，APP冲进App Store前几。

**底层逻辑**：满足用户的「我在故事里」的代入感+「值得分享」的情感价值。

### 洞见5：低俗炒作不可碰

作者明说：「网红不是你想红就能红」，但低俗、暴力、政治红线一碰，**红了也是负能量品牌，寿命有限**。

这跟"懂叔"立场一致：商业洞察、出海经验、新加坡观察可以做，但娱乐化/低俗化擦边球不要碰。

### 洞见6：内容写完先问自己「我会不会转发」

> 你阅读一下，问问你自己会不会转发和分享这样的文章，如此大概就能有个把握。

这是个**极简但极强**的自检工具。比"用户画像""人群标签"这些复杂分析都更接近真相。

### 洞见7：朋友/资源互推的力量

> 平时找一些比较不错的大号互转，不一定粉丝很多，但一定要判断大号的真实性。

不是粉丝多就互推。要看**真实活跃度**——很多"几百万粉"账号实际打开率低于1%。互推选粉丝精准+真实的，比追大V强。

## 📖 值得记录的金句

> 自有平台才是你的用户。凡是你拿不到用户资料的，那是它们的用户，不是你的用户。

> 不是所有用户都有价值，只有目标用户才最有价值。

> 写完之后，你阅读一下，问问你自己会不会转发和分享这样的文章。

> 切忌不可自嗨，沉醉于自己的喜好中不可自拔。要知道这样的文章，百分之百不会受到欢迎。

> 网络世界永远都是虚的，媒体把你捧得越高，当你摔下来的时候，也会越惨。

> 如果你为了钱才创业，那你失败的概率会很大，因为初衷已经走形了。网红炒作亦是如此，一开始就奔着发财的梦去的，往往是发不了财的。

> 一个账号的运营人员更换了，用户一定能看出来。如果换的人不靠谱，这个账号很可能从此就"气绝"了。

> 微博只是平时兴趣，但觉得挺有用的，所以基本上从上午9点发到次日子夜1点都没有停过。
