# 值得信赖的顾问

> David Maister / Charles Green / Robert Galford

> **一句话总结**：建立信任不是性格问题，是可以被学习和管理的系统性过程。赢得客户信任，本质是把关注点从自己身上移开，真正放到客户身上。

## 全书最核心的3个概念

### 1. 信任等式

```
T = (C + R + I) / S

C = 可信度  R = 可靠度  I = 亲近感  S = 自我导向（越低越好）
```

新客户信任值 ≈ 1.25，老客户信任值 ≈ 5，相差**4倍**——这正是开发新客户成本是维护老客户4-7倍的根本原因。

### 2. 信任建立的五阶段

```
委托 → 倾听 → 界定问题 → 构想未来 → 立下承诺
```

最常见的错误：跳过中间三步，从「委托」直接跳到「承诺」，急着给答案。

### 3. 顾问关系的四个层级

```
L1：专业内容提供者（大多数人停留在此）
L2：有能力的顾问
L3：有价值的资源
L4：被信任的专业顾问  ← 目标
```
