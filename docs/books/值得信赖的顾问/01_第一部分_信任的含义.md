# 第一部分：信任的含义（第1-7章）

> **本部分核心**：被信任意味着什么，赢得信任的三个基本技能是什么，以及哪些心态决定成败。

---

## 第1章 全书概览

### 🎯 核心主张
被信任是顾问最有价值的竞争力——信任越深，客户给你的不只是生意，而是对话的权利、容错的空间，和被当作真正伙伴的机会。

### 📌 关键框架
**被信任的16种红利**（客户对你越信任，他们越会）：
1. 主动向你征求建议
2. 真正采纳你的建议
3. 邀请你参与更重要的战略讨论
4. 像你希望的方式对待你
5. 尊重你
6. 分享更多内部信息
7. 不对你的收费锱铢必较
8. 把你介绍给他们的人脉
9. 减少对你工作的干预
10. 相信你的判断
11. 在你犯错时原谅你
12. 在别人攻击你时为你申辩
13. 主动提醒你潜在的风险
14. 让双方都更放松自在
15. 在问题刚出现时就第一时间联系你
16. 相信你对他身边人的评价

### 🔧 可落地的交付物
**客户关系健康度自检**：拿着这16条，对照你目前最重要的3个客户关系，逐条问自己：「这条我有吗？」没有的条目，就是你的信任缺口，也是下一步要做的事。

| 红利条目 | 客户A | 客户B | 客户C |
|---------|-------|-------|-------|
| 主动征求建议 | ✓/✗ | | |
| 真正采纳建议 | | | |
| …… | | | |

缺口最多的客户，优先投入时间。

### 💡 重要洞见
被信任的顾问**不把自己的想法强加于人**，帮客户独立思考而非替他做决定——这正是大多数顾问最难做到的事，因为我们天生习惯给答案。

## 第2章 何为"被信任的专业顾问"

### 🎯 核心主张
顾问与客户的关系分四个层级，大多数人停在前两层——突破到第四层，需要的不只是专业技能，而是关系深度。

### 📌 关键框架

**顾问关系四层级**：

| 层级 | 描述 | 客户找你做什么 | 关键维度 |
|------|------|----------------|---------|
| L1 | 专业内容提供者 | 完成特定任务 | 专业能力 |
| L2 | 有能力的顾问 | 解决更广泛的问题 | 专业能力 |
| L3 | 有价值的资源 | 战略问题、新视角 | 能力+关系 |
| L4 | 被信任的专业顾问 | 几乎任何问题，包括私人问题 | 关系深度 |

两个维度决定层级：商业问题的**广度** + 个人关系的**深度**。

### 🔧 可落地的交付物
**自我定位工具**：选你最重要的一个客户，回答以下问题：
- 他会在周末打电话问你职业建议吗？→ L4信号
- 他会在决策前主动找你讨论，而不是结果出来后通知你？→ L3信号
- 他只在有具体任务时联系你？→ L1/L2信号

如果你大多数客户停在L1/L2，下一步要做的不是提升专业，而是**增加个人关系的深度**——从主动问候、了解对方的非工作处境开始。

### 💡 重要洞见
律师雷吉纳被客户委托帮助妻子面对他即将到来的死亡——这是把生命中最重要的事托付给你。**L4不是职位，是被需要的深度。**

## 第3章 赢得信任

### 🎯 核心主张
信任不是声称出来的，而是用行动赢得的——先付出，后索取。

### 📌 关键框架
**信任的六个特点**：
1. 慢慢培养，没有捷径
2. 一半理智、一半情感——专业实力只是一半
3. 基于双向关系——单方面付出无法产生信任
4. 意味着风险——没有风险的信任毫无意义
5. 对顾问和客户的意义不同——角色决定体验
6. 非常个人——我们信任的是人，不是机构

### 🔧 可落地的交付物
**「直诚时刻」练习**：本周找一个机会，对客户说一件他们可能不喜欢听但真实的事——不是批评，是你真实的判断。观察对方的反应。大多数人会因为你的坦诚而更信任你，而不是相反。

参考「查理和砂纸」：直接说「我们没有这个行业经验」，反而赢得生意。直诚是最快的建信方式，但也是最少人敢用的方式。

**「掀幕布」清单**——每次见客户前问自己：
- 这次对话里，有没有什么「房间里的大象」我一直没说？
- 如果我像律师注意到客户犹豫一样，主动说出来，最坏的结果是什么？

### 💡 重要洞见
**律师和法庭台阶**：律师注意到客户起诉母亲前的犹豫，主动说出「我告诉他，很难想象跟自己的母亲对簿公堂」——客户终止诉讼，此后把所有法律事务都交给了这家事务所。掀开幕布一探究竟，是信任关系的起点。

## 第4章 如何提出建议

### 🎯 核心主张
提建议从来不是纯逻辑活动，而是一场充满情绪的「二重奏」——只懂给出正确答案，而不懂让对方接受，建议等于没有价值。

### 📌 关键框架
**提建议四步骤**：
1. 告诉客户不同的可能性
2. 解释每种可能性（充分讨论）
3. 给出你的推荐
4. **让客户来做出选择**

### 🔧 可落地的交付物
**话术模板**（下次提建议时直接用）：

❌ 不要说：「你必须这样做」/ 「我建议你……」（单向输出）

✅ 改成：「我们来看看有哪些选项。根据我的判断，X方案最有可能解决你的问题，原因是……你怎么看？有没有我没考虑到的？」

**苏格拉底式提问模板**（让客户自己得出结论）：
- 「如果用这个方法，你预期会发生什么？」
- 「你觉得目前最大的障碍是哪里？」
- 「如果可以重来，你会在哪个步骤做不同的决定？」

目标：让客户说出「仔细想来，我觉得B是更好的答案」——那个答案和你原本想说的一样，但他自己想通了。

### 💡 重要洞见
客户承担着个人风险——名誉、晋升、职业前途。这解释了为什么他们有时「不理性地」抵制正确建议。提建议时先处理情绪，再处理逻辑。

## 第5章 建立关系的浪漫法则

### 🎯 核心主张
建立业务关系和谈恋爱遵循同样的原则——把「专业距离」当美德的顾问，往往失去了最关键的竞争力。

### 📌 关键框架
**建立关系的11条原则**：
1. 采取主动——你必须先付出，冒着被拒绝的风险
2. 用行动证明——不是夸夸其谈，是做给他看
3. 发现不同，而非相似点——对方是独一无二的个体
4. 确定客户想要听取你的意见——别总假定他们需要建议
5. 先赢得提建议的权利——明白问题、明白感受、说服他你明白
6. 不断提问——然后闭嘴倾听
7. 心口如一——暗示没用，直接说出来
8. 大胆求助——邀请客户一起解决问题
9. 表现出对他人的兴趣——记住对方说的话，下次提到
10. 赞赏但不奉承——必须基于具体事实
11. 表达感激之情——不要把对方的付出视为理所当然

### 🔧 可落地的交付物
**本周行动清单**（从11条中选3条立刻做）：
- [ ] 主动联系一个最近没有联系的客户，不带任何商业目的
- [ ] 下次见面时，记下客户提到的一件私人事（家人、爱好、近况），下次见面时提起来
- [ ] 找一个机会真诚地感谢客户——具体说出他做了什么、对你意味着什么

### 💡 重要洞见
「专业距离」不是美德，是借口——是顾问在用「职业标准」掩盖对关系风险的恐惧。

## 第6章 心态的重要性

### 🎯 核心主张
技巧只是表层，心态才是根本——五种关键心态决定能否真正建立信任。

### 📌 关键框架
**五种关键心态**：
1. **关注他人**——卡耐基：「影响他人的唯一方法是找出他们的需要，然后告诉他们如何得到它。」（是唯一，不是最好）
2. **自信**——有自信把注意力集中到倾听上，不急着用答案证明自己的价值
3. **内心强大**——「当不在乎功劳归属时，你便能取得更大的成就」
4. **好奇心**——把注意力放在**还未知晓的内容**上，而非已知的事情
5. **包容的专业精神**——与客户协同一致，共同承担责任

### 🔧 可落地的交付物
**会后自检**（每次客户会议结束后用）：
- 这次会议，我说话的时间 vs 倾听的时间，比例是多少？
- 我有没有在客户说完之前就开始想答案？
- 我有没有因为担心「显得无知」而回避某个问题？
- 我今天最关心的是解决客户的问题，还是证明我自己的价值？

**恐惧诊断**：阻止我们关注客户的根本原因是各种形式的恐惧——怕无知、怕失败、怕丢面子。下次发现自己抢着给答案，先问：「我现在在怕什么？」

### 💡 重要洞见
提莫案例：他在高管会议上几乎没发表实质内容，但会议效果极好。他的贡献全部来自对情绪的感知——「乔伊，你不同意这个说法，对不对？」这就是把关注点完全放在他人身上的力量。

## 第7章 是真诚还是伎俩

### 🎯 核心主张
技巧和真诚不是对立的——持续用行动去关心，即使一开始不是真心，长期做下去也会变成真诚。

### 📌 关键框架
两条路都通向信任：
- 从内心出发：真心关心 → 关心的行为 → 信任
- 从行为出发：先做关心的行为 → 开始真正关心 → 信任
- **两端向中间同时走，效果最好**

**如果真的不在乎某个客户，四个选项**：
1. 继续服务但不建立关系（两败俱伤）
2. 继续假装关心（可以试，有时限）
3. **把客户转给同事**（很多时候最优解）
4. **终止合作**（有时唯一正确选择）

### 🔧 可落地的交付物
**客户关系诚实度测试**：
- 你现在的客户里，有没有你其实不在乎、但还在继续服务的？
- 如果有，用上面四个选项判断：哪个对双方最诚实？
- 声誉永远放在收入前面——一个你不在乎的客户，最终会伤害两边。

### 💡 重要洞见
真诚不是天赋，是选择——你选择持续做关心的行为，最终会真的关心起来。

## 第一部分总结

| 技能 | 核心 | 可落地工具 | 最大障碍 |
|------|------|-----------|---------|
| 赢得信任 | 先付出，用行动证明 | 「直诚时刻」练习 | 只想着自己的得失 |
| 有效提建议 | 情感二重奏 | 话术模板+苏格拉底提问 | 急于给出答案 |
| 建立关系 | 主动投入，发现对方独特性 | 本周行动清单 | 把「专业距离」当美德 |

### 关键金句
> 「赢得信任的诀窍是避免一切诀窍。」

> 「勇气，是被信任的顾问与普通顾问之间最真实的分水岭。」
