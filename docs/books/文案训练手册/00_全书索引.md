# 《文案训练手册》读书笔记

> **一句话总结**：文案不是在写字，是在精神上引导一个人从「看见」走到「掏钱」——每个元素都只有一个目的：让他读下一句话，直到他无法停止，直到他买单。

## 全书最核心的3个概念

### 1. 滑梯效应（Slippery Slide）
读者一旦开始读，就无法停下——每一个元素、每一句话，唯一的目的是让他读下一句。广告所有元素→读第一句话→读第二句话→……→买单。这是整本书的底层逻辑。

### 2. 31个心理诱因
人类购买行为背后的心理驱动力，从「参与感」到「诚实」到「恐惧」到「希望」。写文案不是在描述产品，是在触发这些心理机制。

### 3. 以情感卖，以理性诠释
人们因情感购买，然后用逻辑为购买行为合理化。文案的核心任务是触发情感，然后提供足够的理性弹药让买家说服自己。
