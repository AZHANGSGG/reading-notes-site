# 第一部分：做好准备（第1-4章）

> **本部分核心**：在真正带团队之前，先搞清楚自己的角色边界、权责底线、关系基础和激励原理。帮助他人成功，是这个角色唯一的核心职责。

---

## 第1章 明确角色：领导团队者如何创造价值

### 🎯 核心主张
团队领导的工作不是自己干得好，而是让别人干得好——从「为自己负责」切换到「为他人负责」，这是角色的根本转变，也是大多数人没有准备好的转变。

### 📌 关键框架

**团队领导能做的14项工作（时间分配参考清单）**：
1. 非正式与每位成员花时间，帮助其自我成长
2. 了解初级员工职业发展，按其路径调整项目
3. 维护客户关系，监控满意度
4. 帮助成员建立独特价值
5. 帮助成员跟上行业潮流
6. 监控项目盈利性并讨论
7. 建立知识分享机制
8. 帮助探索科技创新
9. 参与招聘
10. 跟踪并帮助成员执行工作计划
11. 组织定期团队行动计划会议
12. 推进正式绩效评估
13. 就绩效进行咨询
14. 庆祝成绩（小胜也算）

**「愿景」vs「让我支援你」**：
- 摩西策略（「跟我冲！」）：大多数人玩不转，只有极少数天赋异禀的人能靠魅力驱动团队
- 真正有效的：「让我支援你」——公仆式领导力，把成就感建立在他人的成功之上

### 🔧 可落地的交付物

**团队领导角色自检（每季度一次）**：

| 工作项 | 上季度做了吗 | 团队感受如何 |
|--------|------------|------------|
| 非正式探访成员（问：我能怎么帮你）| | |
| 了解每个成员的职业目标 | | |
| 庆祝小胜利（不只是大里程碑）| | |
| 帮助成员破除工作障碍 | | |
| 充当传声板（帮人捋清问题）| | |

空白越多，你的领导力缺口越大。

**选择题**：14项工作中你不可能全做，先问自己：
- 哪3项最能提升团队热情和业绩？
- 哪3项团队成员最期望我做？
- 把这3项做到出众，而不是14项都做到普通。

## 第2章 权责分明

### 🎯 核心主张
团队领导最大且最确定的失败，就是没有提前就「我到底应该干什么」达成共识——不谈清楚，注定失败。

### 📌 关键框架

**6项必须事先谈清楚的工作规矩**：
1. **履行领导职责的时间**：最少X小时/年，最多Y小时/年（同时设上下限）
2. **指导员工的权利**：是否有权进行一对一非正式干预和跟进
3. **绩效考核结构和薪酬**：以团队业绩为主要考核依据，个人业绩为辅
4. **对团队成员薪酬的贡献**：正式参与成员绩效评估和薪酬讨论
5. **重视无即时财务回报的时间**：公司要明确认可并考核此类投入
6. **公司管理层的支持**：高管公开表态、参加组会、设立团队领导委员会

**一个反直觉的计算**：让团队领导每周少做500小时客户服务（减少约20万元创收），但他有全部时间管理800万元业务、创造300万利润。20万的「损失」换来数倍回报。很多公司舍不得这笔损失，结果是团队始终低效运转。

### 🔧 可落地的交付物

**上任前必谈清单**：
- [ ] 我每年可用于管理工作的时间，上下限是多少？（已确认）
- [ ] 我是否有权直接与成员进行非正式一对一工作指导？
- [ ] 我的薪酬，有多少比例由团队整体业绩决定？
- [ ] 我是否有机会参与每个成员的绩效评估？
- [ ] 公司管理层是否公开表态支持团队建设？

没有以上共识就开始带团队——你在用信念工作，大概率会失败。

## 第3章 构建关系：关键技能

### 🎯 核心主张
团队领导的影响力来自一件事：成员是否相信你真正关心他们。技巧可以学，「真正关心」是性格——但可以通过行动慢慢培养出来。

### 📌 关键框架

**受信任领导的22个特质**（改编自《值得信赖的顾问》，仅列核心）：
- 毫不费力地理解成员
- 不会朝令夕改，值得依赖
- 帮助创新思考，不代下结论
- 不争功劳，内心强大
- 表里如一，没有装模作样
- 以「爱之深」为前提才「责之切」
- 记住成员说的话（却没刻意记录）
- 从长计议，维护良好关系高于一时麻烦
- 高压之下能幽默化解
- 承认自己的缺点和错误

**「唠叨权」机制**：通过帮助成员建立个人目标，获得定期跟进督促的正当授权。不是强制，是成员自己允许你监督他们完成自己设定的目标。

### 🔧 可落地的交付物

**起始行动计划（新上任三步走）**：

**第一步：每周非正式探访**（不预约，直接进办公室）
- 「最近在忙什么？」
- 「有什么我能帮你的？」
- 约定跟进，言出必行
- 要形成惯例，不能一阵风

**第二步：一对一职业对话**（每位成员，早期优先）
- 三年后你希望在哪些方面鹤立鸡群？
- 你需要公司和团队怎么支持你？
- 你最感恩哪类领导的帮助？
- 迄今为止最大的工作成就是什么？

**第三步：获得「唠叨权」**
- 帮成员确定一个具体的个人目标
- 明确「我可以定期跟你过问这件事吗？」
- 得到许可，才能有效跟进

### 💡 重要洞见
「10-80-10原则」：10%的人天生会做，10%教了也不会，80%是中间地带——通过合理指引，可以学会建立关系和指导他人的技能。对大多数团队领导来说，这是可以习得的能力。

### 📖 值得记录的金句
> 「你的领导力成效，主要取决于你而非团队成员——具体而言，团队成员是否视你为值得信任的提供建议的人。」

## 第4章 勇于激励

### 🎯 核心主张
真正有效的激励，来自于了解每个成员真正想成就什么，然后帮他们看到可能性——而不是愿景演讲，也不是单靠钱。

### 📌 关键框架

**桑德尔实验**：开学第一天宣布「所有人都得了A」，条件是学生当天提交一封次年5月的信，说明「我得到这个A是因为……」。把评分从「记录过去」变成「描绘未来」，彻底改变了课堂关系。

**5种具体激励行动**：
1. 真正关心成员想取得怎样的事业成功
2. 关心成员个人生活中的重大事务
3. 在成员遭遇个人危机或职场危机时出现
4. 时不时非正式地「打个照面」
5. 在成员明显需要帮助时主动伸手（不只是同情）

### 🔧 可落地的交付物

**「打照面」话术**（走廊碰到状态不对的成员时）：
> 「你看起来有点心不在焉，出什么事儿了？」

然后——闭嘴，听他说。不用解决问题，允许他「喷涌而出」，有时候就够了。

**激励月检（每月问自己）**：
- 有没有成员的职业诉求我还不清楚？
- 有没有成员经历个人危机我没有关注到？
- 有没有成员明显需要帮助，但我没有主动伸手？
- 有没有成员的小胜利我没有庆祝？

**弗莱德斯双层对话法**：每次与成员互动都有两层——内容层（说了什么）和信任层（增进还是削弱了关系）。每次对话结束后问自己：「这次对话，对我们的关系有什么影响？」

### 💡 重要洞见与案例
唐·高宁格的故事：总是找机会让下属事业进步，哪怕意味着下属被提拔离开自己的团队。结果他得到了梅斯特见过的最高团队评分，次年继续攀升。——激励别人成功，最终成就了自己。

### 📖 值得记录的金句
> 「任何组织的领导工作都是建立周围人的自信，让这些人都觉得自己身高丈八。」——杰克·韦尔奇

## 第一部分总结

| 章节 | 核心任务 | 最常见错误 | 立刻可做 |
|------|---------|-----------|---------|
| 第1章 | 明确领导者的真正工作 | 继续把自己当个人贡献者 | 14项工作清单，选3项做出众 |
| 第2章 | 上任前谈清楚权责 | 假设大家都懂，从不谈清楚 | 上任前必谈清单6项 |
| 第3章 | 建立信任关系 | 等待关系自然发生 | 起始行动计划三步 |
| 第4章 | 激励每个成员 | 用钱解决所有激励问题 | 激励月检 + 双层对话法 |
