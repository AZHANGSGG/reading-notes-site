# 第二部分 关于客户的问题

> 这是全书最厚实、最实操的一部分，8 章。核心一句话：**专业服务公司的客户工作不是「营销」，而是一整套相互依赖的系统**——倾听 → 超越期望 → 培育关系 → 展开追求 → 广而告之，缺一不可。

## 🎯 核心主张

**专业服务的客户购买行为本质是「在寻找信任」，而不是在比较技术能力。** 进入决赛的几家公司技术上都达标了，最后选谁，看的是谁让客户感觉"踏实、被在乎、可以建立长期关系"。

这一部分的所有方法论——从倾听客户、提供卓越服务、到吸引新客户——都在围绕"如何赢得信任"这一个目的展开。

## 📌 关键框架：8 章逐章拆解

### 第5章 业务开发的一揽子安排（5 大支柱）

| 支柱 | 含义 | 时间分配建议 |
|------|------|-------------|
| **倾听客户需求** | 系统化收集市场情报 | 10-15% |
| **超越客户预期** | 在交付中创造惊喜 | 10-15% |
| **培育客户关系** | 投入老客户的不可计费工时 | 30-35% |
| **展开追求** | 对特定潜在客户的销售 | 25-40% |
| **广而告之** | 行业曝光（研讨会、文章） | ~10% |

**最关键的认知：四象限投资回报率**

| | 已意识到需求 | 未意识到需求 |
|---|---|---|
| **现有客户** | ⭐⭐⭐ 最高 ROI | ⭐⭐ 第二 |
| **潜在客户** | ⭐ 第三 | 最低 ROI |

但绝大多数公司把时间用反了：在"潜在客户 + 已意识到需求"上花最多时间（项目竞标），在"现有客户"上反而最少投入。

### 第6章 倾听客户的需求（7 种方式）

**为什么必须倾听：**
- 缺陷也是财富——不知道哪里没做好就永远没法变强
- 客户的看法是评估当前服务的唯一依据
- 没听到客户意见前的所有"战略思考"都是猜测

**7 种倾听方式（互补而不是互替代）：**
1. **客户反馈小组** — 5-7 名 VIP 客户每年 2-3 次反馈会
2. **反向座谈会** — 邀请客户高管来给我们讲他的行业
3. **参加客户的行业会议** — 听他在自己人面前怎么说
4. **市场调研** — 录像式定性座谈优于纸面问卷
5. **资深合伙人拜访客户** — 餐桌上谈长期，工作室外的层级对等
6. **业务团队的客户反馈报告** — 项目结束时项目负责人坐下来听
7. **客户反馈系统** — 强制性问卷调查，制度化的反馈渠道

### 第7章 工作质量高不代表服务质量好（服务第一定律）

> **服务第一定律：客户满意度 = 客户体验值 - 客户期望值**

两者都是主观的、意识形态的——不仅要做好实质工作，还要管理客户期望。

**修车厂的隐喻：**
- 工作质量 = 把车修好（技术）
- 服务质量 = 修车过程中如何对待车主（体验）
- 客户判断不了技术高低，但能判断态度好坏

**典型的专业服务"以专业人士为中心"的失败：**
- "如果客户不捣乱，我会做得非常出色"——把客户视为干扰
- 建筑师疯狂爱上自己的作品而忽略客户预算
- 律师执着于结案陈词而不顾客户感受
- 顾问痴迷复杂分析而错过客户真正需求

**"成功三大要素"：让客户容易找到你 → 平易近人 → 有能力**（顺序就是这样，不是反过来）

**经典案例（戴瑞·威可夫的酒店）：** 客户随口说窗外景色漂亮，下次来酒店主动安排同一房间——简单的事情但极具杀伤力。

### 第8章 服务质量管理体系（5 个支柱）

| 支柱 | 内容 |
|------|------|
| **跟踪反馈** | 强制性客户满意度问卷（不能替代面对面，但建立责任体系） |
| **管理** | 反馈结果必须有人讨论、跟进、对比团队平均 |
| **方法和工具** | 在每个项目环节找"提升体验机会点"，写进操作手册 |
| **培训** | 客户沟通技巧不是天生的，可以训练 |
| **奖励** | 第二年开始把客户满意度纳入合伙人薪酬 |

**关键洞见：调查问卷的真正作用不是"测量满意度"，而是建立责任意识。**

详尽的 25 题问卷（评分 1-5）才是真诚的——问得越具体，客户越觉得你是在认真听。简单几个问题反而显得敷衍。

回复率通常超过 75%。客户拒绝是少数特例（<1%）。

**警告：千万不要在质量反馈问卷里塞营销问题（"你还需要哪些新服务"），客户立刻识破你的目的。**

### 第9章 针对现有客户开展市场营销

**为什么现有客户是金矿：**
1. 成功率更高（已有信任基础）
2. 营销成本更低（已了解客户）
3. 利润更高（不竞标、可以加杠杆）
4. 能积累新能力（客户更愿意让你做新业务）

**为什么大多数公司忽视现有客户：**
- 开发新客户更刺激（狩猎快感）
- 公司奖励制度只对新客户庆祝
- "对老客户应该收费"——会计制度阻碍
- 老客户营销不像新客户那样有清晰流程
- 营销活动是被动响应招标

**3 步走（缺一不可）：**

**Step 1：让客户乐意再次聘用你**
- 当前项目超越预期
- 增加接触机会
- 主动参加客户内部会议（最高 ROI 技巧）
- 建立业务关系 > 个人关系

**Step 2：提高公司服务客户的能力**
- 投入时间了解客户行业、业务、组织、人员
- 这一步既能服务好当前项目，又是为下一个项目做铺垫

**Step 3：寻找并努力赢得下一个项目**
- 前两步做好了，第三步水到渠成

**关键 trick：把"开发某客户"作为独立预算科目**——不可计费工时变成"营销投资"，初级员工也能记账参与。

### 第10章 客户如何做选择

**两阶段：资格认定 → 做出选择**
- 资格认定：理性的，看你的能力、案例、声望
- 做出选择：感性的，看人际感觉、信任、风格

**到了决赛阶段，技术能力差异不重要——客户在挑「想和谁合作」**

**购买者的 5 种情绪（专业服务购买经常不愉快）：**
1. **冒险感** — 把事情交出去意味着放弃控制
2. **忐忑** — 不知道自己问题简单还是复杂、谁最合适
3. **怀疑** — 你会真心了解我，还是用行话忽悠？
4. **被曝光感** — 我的"家丑"会被你发现
5. **威胁感** — 你的发现可能会暴露我做得不好

**赢得信任的核心方法：**
- 提前做功课（看年报、了解他的位置和角色）
- 把陈述句改成问句："我们觉得有 3 个趋势，你同意吗？"
- 用「投之以桃」策略——先免费给客户提供有用建议
- 教客户一些他不知道的事
- 对客户提的异议先承认其合理，再回应
- 不要替客户决定方案，给他选项让他选

**最重要的反直觉认知：**
> **聘用决定通常在写项目建议书之前就形成了。** 正式的演示和方案陈述只是确认（或推翻）一个已经形成的决定。

**演示陈述的真正考验不是流畅，而是临场应变能力。**

### 第11章 吸引新客户（首选 4 大手段 + 替补）

**4 个营销原则：**
1. **果酱法则**——撒得越广越薄，集中精力在少数目标客户
2. **证明 > 宣称**——用作品说话，而不是说自己有多好
3. **面对面 > 书面**
4. **循循善诱 > 咄咄逼人**

**首选营销手段（按效果排序）：**

| 手段 | 关键要点 |
|------|---------|
| **小型研讨会** | <25 人，专业服务公司:潜在客户 ≤ 1:5，提前研究每位客户8页背景 |
| **演讲** | 不是给所有听众留印象，而是筛选当下需要服务的人。脱稿、互动、答问 |
| **分享文章** | 在客户关注的刊物（不是行业刊物）发表，2000-5000 字模板 |
| **专项调研** | 提供其他竞争对手没有的信息，最好行业协会赞助 |

**替补手段（不能替代首选）：**
- 社区服务/民间组织职位
- 行业简讯（容易雷同，效果一般）
- 媒体宣传（封面人物只让你妈高兴）
- 宣传手册（违反"多说好处少展示特点"原则）
- 大型研讨会、电话推销、广告（基本不奏效）

**只有写得有真实案例、有具体内容的视频宣传资料才有效。**

### 第12章 管理营销投入

**核心矛盾：知道该做什么，但做不到。**

**问题根源：营销是"不可计费工时"的额外活动，可计费工时清晰可见，营销活动则杂乱无章。**

**主要管理误区：**
- 只奖励"营销结果"——不奖励"营销投入"——所以无人愿意做铺垫工作
- 奖励新客户多于老客户——所以老客户被忽视
- 让每个合伙人自己决定参不参与——所以没章法

**解决方案：管理营销「投入」而不是「结果」**

**核心机制：小团体协作**
- 每个团队 3-5 人，自愿组队（不能指派）
- 每团队选定一个目标行业/领域
- 每季度一个规划期，三分之二时间预先安排，三分之一留给意外
- 每位合伙人有最低营销时间标准
- 团队定期在合伙人会议上汇报进展（不是结果）

**初级员工是营销富矿：**
- 撰写文章、准备研讨会、调研客户需求都不需要"销售天分"
- 现有客户营销天然适合初级员工——他们和客户接触最多

**避免"行业简讯"误区：** 客户都在抱怨"不要告诉我你们公司有什么动态，告诉我对我意味着什么"。

## 💡 重要洞见与案例

### 洞见1：服务第一定律的力量

**客户满意度 = 客户体验 - 客户期望**

这个公式的真正威力在于：**两边都是主观的**。所以：
- 餐厅说"等位 20 分钟"实际 15 分钟 → 惊喜
- 餐厅说"等位 5 分钟"实际 10 分钟 → 不满
- 同样是 10 分钟体验，结果完全不同

### 洞见2：90% 的推荐与技术质量无关

梅斯特的调研发现：客户推荐供应商时，仅 10% 是因为技术质量好，90% 是因为关系管理好。

技术过硬只是入场券。真正决定推荐和复购的是软实力。

### 洞见3：购买者的"五种情绪"

客户聘请你时是在冒险、忐忑、怀疑、被曝光、被威胁——这跟你想象的"客户对我们感兴趣"完全不同。

每次见客户前先想"他现在最焦虑的是什么？"，比想"我要怎么展示自己"管用得多。

### 洞见4：聘用决定 ≠ 项目建议书决定

合同决定通常在准备项目建议书 BEFORE 就形成了。建议书只是确认或推翻。所以：
- 写得最好的建议书也救不了之前没建立的关系
- 演示阶段重要的是"答问题"，不是"讲得流畅"

### 洞见5：投之以桃报之以李

最强的销售策略：**先免费帮客户**——发现一个他没看到的机会、发一篇相关文章、引荐一个合适的人。免费帮过他三次之后，第四次他自然来找你。

### 洞见6：业务关系 ≠ 个人关系

很多人以为"和客户成朋友"就能赢——错。建立**业务关系**（让他放心你的工作、信任你的判断）比**个人关系**（一起喝酒打球）重要得多。

朋友关系是业务关系的补充，不能替代。

### 洞见7：研讨会的 1:5 法则

潜在客户 / 公司人员 ≤ 5:1，总人数 ≤ 25。
否则就是公关活动，没有业务效果。

### 洞见8：临时抱佛脚法则

大多数专业服务公司在做的：媒体宣传、宣传手册、大型研讨会、冒昧电话——梅斯特统称"临时抱佛脚"，效果差。

**唯一有效的 4 个：小型研讨会 + 客户行业会议演讲 + 客户行业刊物文章 + 专项调研。**

### 洞见9：重要的不是「你做什么」，是「你怎么做」

> "重要的不是你做什么，而是你如何去做，这样才会有成果。"

服务行业的核心定理。
