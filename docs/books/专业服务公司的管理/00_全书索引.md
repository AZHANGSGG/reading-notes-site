# 《专业服务公司的管理》全书索引

---

## 🎯 一句话核心主张

**专业服务公司本质是个特殊的资产负债表——同时管「利润表」（收费、利润）和「资产负债表」（技能、客户关系）。所有失败案例的根源都是只看利润表、不看资产负债表。**

围绕这一中心，全书 32 章给出了 7 大维度的系统答案。

## 🧭 核心概念（必背）

### 1. 杠杆率（Leverage）
不同层级专业人员在项目上的平均时间投入比例。决定：客户服务能力、员工职业发展、合伙人利润。**整本书的地基概念。**

### 2. 三种服务类型
- **专家型（Brain）**：卖创新和前沿——低杠杆、价值定价、个人品牌
- **经验型（Gray Hair）**：卖判断力和经验——中等杠杆、工时计费、机构品牌
- **程序型（Procedure）**：卖效率和成本——高杠杆、固定价格、流水线

### 3. 三种角色
- **找项目的人**（资深合伙人）
- **管项目的人**（经理）
- **做项目的人**（初级员工）

### 4. 服务第一定律
**客户满意度 = 客户体验 - 客户期望**
两边都是主观的，所以管理预期和提升体验同等重要。

### 5. 健康因素 vs 卫生因素
- **健康因素**：长期根本性（涨价、加杠杆、提升能力）
- **卫生因素**：短期效果（减开支、提利用率）
绝大多数公司把时间花在卫生因素上。

### 6. 系统性授权不足
专业服务公司的"职业病"——典型公司 40-50% 的资深时间花在初级员工就能干的活上。

### 7. 个人资产负债表
专业人士的两组核心资产：
- **资产A：知识与技能储备**（贬值速度比想象快）
- **资产B：客户关系与声誉**（深度比数量重要）

### 8. 动力循环
高动力 → 高效率/质量 → 市场成功 → 经济效益 → 丰厚回报 → 高士气 → 高动力。**正向逆向都是循环——成功孕育成功。**

### 9. 教练理论
管理 = 教练。两句话：
> "对，你很厉害……但你到底有多厉害呢？"

第一句满足虚荣心，第二句制造下一个挑战。

### 10. 农民公司 vs 猎人公司
- 农民：集体决定种什么庄稼，全员押注（团队作战、统一服务）
- 猎人：每人去打猎（个人主动性、灵活反应）
- **二者必择其一，不可兼得**

### 11. 一体化公司（"One-firm" Firm）
高盛、麦肯锡、安达信、翰威特、瑞生的共同模式：
SWANs 招聘 + 严格培训 + 自己培养 + 控制速度 + 挑剔业务 + 不弃员工 + 评判薪酬 + 研发投入

### 12. 7 大资产管理方法
1. 回顾性综合评价
2. 对技能转移行为衡量奖励
3. 对客户满意度衡量奖励
4. 资产负债表问题进考核
5. 现有客户营销
6. 项目系统化总结
7. 项目方法论的工业工程化研究
