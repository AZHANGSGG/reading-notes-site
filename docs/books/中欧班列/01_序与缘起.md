# 01 序与缘起 — 古丝路到新班列

## 🎯 核心主张

**中欧班列不是一个铁路项目，是全球供应链的"第三选择"试验场。** 它在海运（45天）和空运（快但贵到只能装高值小件）之间，开辟了一条16-18天、成本是空运1/4的中等通道，第一次让"内陆出海"成为可能。

## 📌 关键数据骨架

| 维度 | 数据 |
|------|------|
| 首发 | 2011年3月19日，重庆 → 杜伊斯堡 |
| 增长 | 2011年17列 → 2018年6300列（370倍） |
| 累计 | 截至2018年累计开行超12000列 |
| 时间 | 海运45天 vs 班列16-18天（缩短2/3） |
| 时间优化 | 全程运行时间从初期20天缩短到14-16天 |
| 成本 | 整体运输费比初期下降40% |
| 中欧贸易 | 2018年中欧双向贸易超6000亿美元（大部分仍走海运） |
| 长期目标 | 德国物流协会预计2030年欧亚陆路铁路运量将达95万标箱 |
| 网络 | 截至2018年覆盖15国49城，国内56个开行城市 |

## 🧱 三段历史的对照

| 时段 | 主导方式 | 通道 | 商品 |
|------|---------|------|------|
| 古丝绸之路 | 骆驼商队 | 长安/敦煌 ↔ 中亚 ↔ 欧洲 | 丝绸、瓷器换金银宝石 |
| 大航海后 | 海运 | 西非好望角 / 地中海 → 欧洲 | 大宗工业品（被海运垄断300年） |
| 21世纪 | 海运+空运+**班列** | 中国西部 → 阿拉山口/霍尔果斯 → 中东欧 | 笔记本电脑、汽车、电子、葡萄酒、奶粉 |

历史的经验教训：**"层层的盘剥早已演变成合法的服务业，原本蕴含巨大利益的中间环节被大大削减。"** 班列不是要取代海运，是要在中间环节里抢回属于陆上贸易的合理空间。

## 🔧 可落地的交付物

### 「三种运输方式选择决策表」（给客户用）

| 维度 | 海运 | 中欧班列 | 空运 |
|------|------|---------|------|
| 时间（中国-欧洲） | 35-45天 | 14-18天 | 1-3天 |
| 单位成本 | 最低 | ≈ 海运1.5-2倍 | ≈ 班列4倍 |
| 适合品类 | 大宗、低值、不急 | 电子产品、汽车配件、葡萄酒、奶粉、化工 | 高值小件、急件、生鲜 |
| 单批量级 | 集装箱级 | 集装箱级 | 公斤级 |
| 关键风险 | 时间长、汇率波动 | 单点拥堵、补贴退坡 | 价格高 |
| 最适合什么样的企业 | 传统制造业出口 | 中西部制造业 + 欧洲中东部市场 | 电商小件、奢侈品 |

### 「客户决策三问」

要给客户判断要不要用班列，问三个问题就够：
1. **保质期 / 货值 / 资金占用成本** —— 多1个月在路上，能不能承受？
2. **目的地是不是在班列覆盖的"蓝香蕉地带" / 中东欧 / 中亚?** —— 不在覆盖范围里硬上班列没意义。
3. **能不能凑够整箱?** —— 拼箱也行，但单件零散用班列性价比不如DHL。

## 💡 重要洞见

### 1. 班列是被"客户痛点"拉出来的，不是被规划出来的

最早的渝新欧是因为台资笔记本电脑代工厂广达落户重庆后发现：
- 笔记本零部件价格波动大、汇率波动大
- 海运一个月在路上，价格定不下来
- 重庆是内陆山城，海运得先到沿海再走，物流成本和时间都不行

广达的痛点 → 重庆市政府解决方案 → 推动了整个铁路大通道的诞生。

**这是一切B2B生意的真理：通道不是从政策里长出来的，是从一个具体客户的具体痛点里长出来的。**

### 2. "鸿海集团（富士康）2008年第一个吃螃蟹"

测试班列从深圳出发，取道蒙古，跨西伯利亚铁路驶向东欧——比"渝新欧"还早3年。富士康一贯低调，没有任何新闻报道这事，但它**用真实业务验证了陆运可能性**，给后来者打开了想象。

### 3. "中欧班列像一个襁褓中的婴儿"

作者自序原话。这个比喻很重要：
- 婴儿期：补贴扶持、政策呵护
- 但作者明确说"她的健康成长需要朋友的陪伴"——意思是**通道不能只靠政府养，要靠市场参与者的生态**

补贴是早期助产士，不是长期奶妈。**这个判断是后面第5章"补贴退坡"思路的基石。**
