# 03 发展进程与铁海联运 — 蓉欧+战略 + 中远海运

## 🎯 核心主张

**班列做大之后，单纯做"运输"会被竞争挤死，必须升级成"通道服务商"。** 第3章用成都和中远海运两个标杆案例，讲清这个升级路径——从开行一条线，到搭建一张网；从送货，到帮客户打通整个供应链。

## 🧱 「蓉欧+」战略的三圈

成都的核心创新是把"班列"变成"+战略"，意思是不只做开行，做"以班列为锚"的整个生态：

```
              中欧 / 东欧 / 西欧
                     ↑
                     ↑
                  波兰罗兹
                     ↑
                     ↑
           成都国际铁路港 ←─────── 长三角 / 珠三角 / 环渤海
                     ↓                       (面向泛亚地区)
                     ↓
              日本 / 韩国 / 东南亚
```

- **西向**：成都 → 罗兹/波兰，再辐射西欧
- **东向**：成都接长三角、珠三角、环渤海产业链，做泛亚集散
- **南向**：通过铁海联运到东南亚和日韩

**这是把"线"变成"网"的关键一步。** 一条班列就是一根针；把针穿上线，再织成网，整个城市的产业整合能力就完全不一样。

## 🔧 可落地的交付物

### 「通道公司升级三段论」

借成都案例，物流通道公司的成熟路径：

| 阶段 | 角色 | 收入来源 | 关键能力 |
|------|------|---------|---------|
| 1.0 运输公司 | 发车 | 运费差 | 火车头、集装箱、驾驶员 |
| 2.0 通道运营商 | 调度 | 平台费 + 货代分润 | 海关协调、IT系统、回程货匹配 |
| 3.0 供应链服务商 | 整合 | 一揽子服务 | 海外仓、配送、组装、关税优化、金融 |

成都从1.0做到2.5；荷兰物流业从世界各地集货到荷兰组装再分销欧洲，是3.0标杆。

### 「铁海联运四种典型模式」（给客户讲组合拳）

中远海运在书中给出的几种"逆向反馈"组合：

1. **班列去 + 海运回**：东边发欧洲走班列（快），回程中国货走海运（便宜）
2. **海运去 + 班列回**：紧急回程货物走班列（如圣彼得堡 → 中国西部）
3. **铁海联运到第三国**：成都 → 鹿特丹 → 海运到英国/爱尔兰/挪威/葡萄牙
4. **多式联运闭环**：船公司+码头+铁路+卡车一体，给客户"门到门"

**这4种模式是给客户做物流方案的"乐高"，而不是只有一条死板的线路。**

## 💡 重要洞见

### 1. "蓝香蕉地带"是最值钱的5个字

欧洲中西部从英国曼彻斯特一路向南，经过荷兰、比利时、德国西南部、瑞士，到意大利北部米兰、都灵——这条带：
- 1.1亿人口
- 欧洲人口最密集、购买力最强
- 贡献欧洲GDP增长的60%

**对中国出海企业来说，这就是欧洲的"必争之地"。** 看一家中国企业的欧洲战略是否清晰，就看它有没有把"蓝香蕉"作为锚点城市矩阵。

### 2. 鹿特丹港的1/4集装箱来自中国

这个数据很重要——它说明：
- 中国制造已经深度嵌入欧洲供应链（不是"在追赶"，是"在主导")
- 班列再怎么涨，海运基本盘动不了
- 但反过来，**1/4市场份额对应的服务、清关、仓配、保险、关税服务市场，是中国物流公司在欧洲扩张的真正机会**

不是去抢海运的存量，而是抢海运服务链上的增量。

### 3. 荷兰是中国企业进入欧洲的"门户"

为什么不是德国？德国是制造业大国，但**荷兰是物流和金融大国**。中资企业落户欧洲：
- 2017年底荷兰超550家
- 2011年单年就有30家进驻
- 涵盖金融、运输、农业、贸易

荷兰胜出的原因：
- 鹿特丹港 + 史基浦机场（多式联运）
- 蓝香蕉的入口
- 营商环境（语言、税收、法律）
- 物流业极度发达（欧洲一半第三方物流是荷兰公司做的）

### 4. 中远海运2018年开"中蒙俄"班列的"逆向反馈"逻辑

中远海运不是铁路公司，它是海运公司，但它做了班列——为什么？
- 客户在欧洲，集装箱回不来
- 通过班列把空集装箱从圣彼得堡运回中国，重新接入海运体系
- 形成传统供应链的"逆向"反馈环

**这是供应链管理里最高明的一招——不是单点优化，而是闭环优化。**
