# 《做课》读书笔记

> **一句话总结**：课程设计不是玄学而是科学——用产品经理思维把经验结构化，任何有3年以上经验的职场人都可以做出80分以上的好课。

## 全书最核心的3个概念

### 1. 知识资产三层次（决定你能做什么课）

```
Level 1：重要项目经验  →  适合知识小课
Level 2：重复工作方法论  →  适合知识小课 / 体系课程
Level 3：直觉判断决策  →  适合体系课程（最高价值，最难萃取）
```

### 2. 双钻模型四步流程

```
盘点知识资产（发散）→ 设定教学目标（收敛）→ 确定课程结构（发散）→ 设计学习体验（收敛）
```

每步都可以反复调整，不是线性单向的。

### 3. 截屏点「1234」

让学员忍不住截图保存的地方——每张内容页必须是以下之一：
- **1清单**：步骤/自检/注意事项
- **2对比**：两种方案/新旧对比
- **3支柱**：三要素关系（维恩图/三角图）
- **4象限**：两维度判断决策

检验标准：截图发给没看过这门课的人，10秒内能看懂吗？
