# 第四章：如何生产一门体系课程

> **本章核心**：体系课程是讲师的「产品」——做之前先过三圈理论的关，做的过程中靠用户调研、竞品分析和交付物打磨保证质量。

---

## 🎯 核心主张
体系课程的生产成本高、周期长（8小时课至少耗时1个月），做之前必须先判断这个方向值不值得做——三圈理论是那道关。

---

## 📌 关键框架

### 三圈理论：判断是否值得做

值得做体系课程的方向，必须同时满足三个条件：

| 维度 | 问题 | 判断标准 |
|------|------|---------|
| 讲师擅长 | 我对这个方向有久经考验的实践积累吗？ | 有多个成功案例 + 曾指导过他人 |
| 用户刚需 | 学员真的需要这个课程吗？ | 与业绩相关度高 + 工作中频繁发生 |
| 市场竞争力 | 我的课程有差异化切入点吗？ | 人群/内容/目标/结构上的差异 |

> 市场上没有同类课，不代表值得做——可能是需求本来就小。

### 三种体系课程结构

| 结构 | 适用场景 | 典型特征 |
|------|---------|---------|
| 流程串联式 | 能力提升课，知识有上下游 | 先学A才能学B，线性递进 |
| 模块并列式 | 岗位体系课，能力相对独立 | 每章可单独成立，模块并列 |
| 循序渐进式 | 方法论应用于不同场景 | 一套方法从易到难，考验全局设计 |

### 交付物三标准（做课程结构时必须对照）

优质体系课程的交付物必须满足：
- **方法论扎实**：流程明确，每步可被检验
- **交付很清楚**：学员学完可以把交付物做出来
- **结果可衡量**：有明确的自检标准，学员知道合不合格

---

## 🔧 可落地的交付物

### 三圈理论评估表（做课前必填）

```
课程方向：_______________

维度一：讲师擅长
□ 我在这个方向有3个以上的成功案例
□ 我的方法论已经被验证过（不只是做过，而且有稳定成果）
□ 我曾用这套方法指导过别人，并且他们也成功了
→ 如果三项都勾上，这个维度通过

维度二：用户刚需
□ 这门课解决的问题与学员的业绩直接相关
□ 这类问题在学员工作中至少每月都会遇到
□ 我做过至少5个目标学员的一对一访谈，确认了这个需求
→ 如果三项都勾上，这个维度通过

维度三：市场竞争力
□ 我找到了一个与现有课程不同的切入点（人群/内容/目标/结构）
□ 这个差异点是我真正擅长的，而不只是「别人没做过」
→ 如果两项都勾上，这个维度通过

三个维度都通过 → 值得做
只有两个维度通过 → 重新找切入点
只有一个维度通过 → 这个方向暂时放下
```

### 用户调研访谈问题模板（做体系课程前必做，至少10人）

**学员身份背景**：
- 你现在在什么岗位，主要负责什么？
- 你在这个岗位上工作了多久？
- 你们公司大概有多少人？

**具体工作任务**：
- 你每天/每周最核心的工作任务是什么？
- 你的工作成果是用什么指标来衡量的？
- 你最常用的工具和方法是什么？

**面临的挑战**（最重要）：
- 你工作中最常遇到的困难是什么？（让他说具体的，不是泛泛的）
- 这些问题多久会发生一次？（判断刚需程度）
- 你现在用什么方法解决这些问题？（找现有解决方案的缺口）
- 如果有一门课能帮你解决X问题，你觉得应该教什么？

### 竞品分析表（做课前填）

| 竞品课程名 | 目标学员 | 核心内容 | 交付程度 | 学员评价（好） | 学员评价（差） | 我的差异点 |
|----------|---------|---------|---------|------------|------------|---------|
| | | | | | | |
| | | | | | | |

**差异化方向**（四选其一或组合）：
- 人群差异：更细分的人群（行业/公司规模/工作年限）
- 内容差异：更体系化，或更垂直细分
- 目标差异：比竞品多交付一个程度（从「深刻理解」到「简单应用」）
- 结构差异：更好的学习设计，更独特的案例

### 体系课程自检清单

**前期（开始做之前）**：
- [ ] 三圈理论三个维度都通过了吗？
- [ ] 做了至少10场一对一用户访谈？
- [ ] 竞品分析完成，差异化切入点找到了？

**结构（做课过程中）**：
- [ ] 章与章之间有清晰的逻辑关系？
- [ ] 每节都有对应的核心交付物（工具/模板/清单）？
- [ ] 难易章节做了穿插设计，不会持续扎堆高难度？
- [ ] 每个交付物都满足「方法论扎实、交付清楚、结果可衡量」三标准？

**内容（做完之后）**：
- [ ] 课程里所有方法论，我自己都多次实践验证过？
- [ ] 需要练习题的章节（熟练掌握程度），练习题设计完成了？
- [ ] 优秀范例准备好了（可以是自己的，后续替换成学员优秀作品）？

---

## 💡 重要洞见
体系课程生产过程中，**打磨交付物**是最关键的动作——如果交付物想清楚了，内容细节和PPT设计都很容易；如果没做好，做到一半发现跑偏，推倒重来的概率极大。
