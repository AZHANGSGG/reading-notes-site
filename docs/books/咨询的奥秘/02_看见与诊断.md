# 第二部分：看见与诊断——顾问最核心的能力

涵盖：第3章「在不知道自己在做什么时保持高效」、第4章「看到那里有什么」、第5章「看到那里没什么」、第6章「避免陷阱」

---

## 🎯 核心主张

**顾问被请来的时候，问题往往超出你的专长——会做事的顾问不是更专业的专家，而是**更会"看"**的人。会看的能力包括三层：看到那里实际有什么（历史、标签背后）、看到那里没什么（缺失的工具/历史/求助）、以及在自己快犯本不该犯的错误时给自己设个提示物。**

---

## 📌 关键框架

### 1. 马文医学六大秘密（处理任何"自愈型系统"的用药指南）

| 秘密 | 内容 | 顾问业务对照 |
|------|------|-------------|
| 头号 | 90%的病都会自愈 → **要温柔地对待能够自愈的系统** | 等于工程第一法则"没坏就别修"。客户的小问题不要急于干预 |
| 第二 | 反复治疗一个能自愈的系统会最终让它不能自愈 | 客户养成依赖 = 你的资产；客户失去自我修复能力 = 你的负债 |
| 第三 | 每个处方包含两部分：药品 + 正确使用方法 | 单纯交付方案不够，必须配合**怎么用**的说明书；客户不签跟进协议很容易吃错药 |
| 第四 | 已经做过的事情没解决问题，就告诉他们做点别的 | 克朗凯特医生疗法。陷在一个方法里出不来的客户，给个完全不同的方向就行 |
| 第五 | 让他们付足够的钱，他们才会照你说的去做 | **咨询中最重要的活动就是开出正确的价码**（详见第12章） |
| 第六 | 来得早不如来得巧 | 同一个建议，时机不同效果差10倍 |

### 2. 伯登 / 镀金 / 逆镀金 三法则（怎么处理"缺陷"）

```
伯登法则：要是你不能改掉缺点，就把它变成特点
   ↓
镀金法则：要是没法当成特点来宣扬，那就冒充一下（裁缝列维 → 跛行专家）
   ↓
逆镀金法则：所有镀金的东西都得改正（"成本-效益分析"=成本分析 / "灵活性"=想开就开）
```

**这三条相互制衡**：
- 伯登法则是"利己但不损人"的——给客户加分
- 镀金法则是危险的——糊弄过去就不再学习
- 逆镀金法则是顾问的"X光"——客户用什么委婉语，背后就藏什么真问题

### 3. 锤子定律（专业工具的诅咒）

> 圣诞节收到一把锤子的孩子会发现所有东西都需要敲打。

### 4. 伯丁追溯原理 + 斯巴克斯解决问题定律 + 研究指南

```
伯丁追溯：事情是一步步变成现在的样子的
       ↓ 所以要研究历史
斯巴克斯：你越接近找出造成问题的人，解决问题的机会就越小
       ↓ 所以研究历史时不能批评
研究指南：
  • 保持简单，不要太详细——你是顾问，不是律师
  • 研究是为了理解，不是为了批评
  • 在现状中寻找你喜欢的东西并加以赞美
```

**威尔弗兰的白面包故事**：奥登豪瑟夫人当年瞧不起惠灵顿夫人的难吃面包→自己开店→走完了同一条工业化变难吃的路。她不肯研究历史，注定历史重演。  
**白面包警告**：如果你用同样的做法，就会得到同样的面包。

### 5. 标签法则 + 五分钟法则 + 三个手指法则（听客户语言的三件套）

| 法则 | 内容 | 用法 |
|------|------|------|
| 标签法则 | 我们大多数人买的是标签，不是商品；事物的名称并不是事物本身 | 不要被客户给问题贴的"标签"绑架（"维护问题"其实是设计问题） |
| 五分钟法则 | **客户永远都知道怎么解决自己的问题，并且会在头五分钟里讲出来** | 第一次见客户的前5分钟必须全神贯注，事后录音回放 |
| 三个手指法则 | 你用一根手指指别人的时候，看看另外三根手指在指哪里 | 客户激动地指责某人/某部门时，注意他自己藏了什么 |

**误导法的识别**：同一个事实，"超支"vs"预算不足"。管理者说超支保护自己，员工说预算不足把矛头转回管理者。**两个标签都在隐藏真相**。

### 6. 看到那里**没什么**的工具箱（更高级的能力）

| 缺什么 | 提示什么 |
|--------|---------|
| 没有展示投诉的工具 | 这个组织有质量问题但还不知道 |
| 没有突出的重大问题（平均定律） | 已有有效的问题解决机制——可以借力 |
| 有突出的大问题 | 缺少"按严重性排序"的方法和策略 |
| 没人主动求助 | 即使你恰好在场，也不要不请自来 |
| 没有女性 / 没有35-50岁的人 / 没人提交付日期 | 文化或治理结构的盲区 |
| 私人物品全无 | 团队没有归属感 |
| 没人擦黑板 | 工具用法变形（私人公告板而非协作工具） |

**找出"少了什么"的5个方法**：
1. **意识到自身局限**：把客户请求变成书面的，对照核查
2. **利用他人**：问清洁工/外行/不同层级的人——他们看见你看不见的
3. **研究其他文化**：丹麦的"午餐小姐"vs美国各自散去
4. **洗衣单**（vs检查单）：洗衣单提醒**可能**忘记的；检查单要求**必须有**的——前者更适合不清晰的诊断
5. **检查流程**：列出你想到的清单vs洗衣单的对比，差异处=你的盲区

### 7. 三的法则 + 温伯格离奇定律 + 对不一致的洞察力

| 法则 | 内容 |
|------|------|
| 三的法则 | 要是你想不出计划中可能出现的三处问题，你的思维肯定是哪里出了问题 |
| 温伯格离奇定律 | 有时候觉得离奇只是因为眼界不够（同父同母同日所生不是双胞胎——是三胞胎） |
| 对不一致的洞察力 | 语言和音乐不搭配的时候就指出了一个缺漏 |
| 布朗的天才遗产 | 语言经常很有用，但听听音乐总有好处，特别是你自己内心的音乐 |

### 8. 避免陷阱：泰坦尼克效应 + 主管线格言 + 警钟系统

> **泰坦尼克效应**：以为灾难不可能发生往往会导致不可想象的灾难  
> **主管线格言**：你不知道的东西可能不会伤到你，但你记不得的东西肯定会  
> **薯片原理**：如果你了解你的受众，就很容易设置提示物

## 🔧 可落地的交付物

### 工具1：诊断新客户的"五分钟前 + 五分钟中 + 五分钟后"清单

**前**（接到询盘到第一次见面之间）：
```
[ ] 让对方用一封短信/邮件复述一遍问题（书面化）
[ ] 在邮件里看：客户给问题贴了什么"标签"？背后真问题可能是什么？
[ ] 客户用了哪些"镀金词"？（"成本-效益"、"灵活性"、"战略调整"等）
[ ] 这是客户主动找的我，还是别人介绍的？（咨询第四定律）
```

**中**（前5分钟）：
```
[ ] 全神贯注，不打断（哪怕觉得很想打断）
[ ] 录音（征得同意），事后回放
[ ] 听语言 vs 听音乐：哪几个话题语气紧张/松弛？是否一致？
[ ] 关注客户提到的"标签"和"借口"
[ ] 留意客户挥手指别人时，剩下三根手指指向自己的什么
```

**后**（结束5分钟内）：
```
[ ] 立刻写下：客户最想解决的问题（按重要性，3条）
[ ] 写下：你感觉到的不一致点（情绪 vs 内容）
[ ] 写下：客户没说但你直觉很重要的东西
[ ] 不在24小时内提方案——避免锤子化
```

### 工具2：缺失诊断洗衣单（接每个新客户必走一遍）

```
组织缺什么？
[ ] 有没有质量/投诉跟踪机制？
[ ] 有没有人员定期休假和后备替代？
[ ] 有没有女性/不同年龄层/不同背景的人？
[ ] 私人物品在办公室是否可见？
[ ] 谁来真正负责讲交付日期的话？
[ ] 有没有人主动求助？还是被迫的？

人际缺什么？
[ ] 有没有不同层级的人愿意单独和我说话？
[ ] 有没有"清洁工视角"的人？
[ ] 有没有外部审计/独立董事/外部顾问的角色已经在场？

流程缺什么？
[ ] 决策时有没有非正式的"共餐场域"？（丹麦午餐小姐）
[ ] 黑板/白板被使用还是被霸占？
[ ] 文档审核是有人做还是只挂个名？

我自己缺什么？
[ ] 我对这个客户/行业的真实经验有多少？哪些是装出来的？
[ ] 我现在在用哪把"锤子"？这把锤子合适吗？
[ ] 我有没有需要的人脉（丹麦的午餐小姐、本地律师、税务专家）来补我的缺？
```

### 工具3：橙汁测试 + 折中图 + 三的法则 → 报价框架

接到客户报价请求时，用一个三步框架：
```
Step 1【橙汁测试】给一个具体苛刻的要求，看对方能否回答"我们能做，这是费用"
        ↓
Step 2【折中图】把客户最高级的形容词（最快/最便宜/最好）画成二维图
        ↓ 让客户自己挑要牺牲哪一边
Step 3【三的法则】列出可能出现的3处问题——如果列不出，你的方案有盲区
        ↓
        然后给报价
```

### 工具4：克朗凯特医生疗法（卡住的客户的解药）

> 已经做过的事情没解决问题，就告诉他们做点别的。

**用法**：当客户说"我们试过A，没用"——直接说"那试试B"，不需要先论证为什么B会work。  
**注意**：必须配合第五大秘密（要价足够高），否则客户不会真的去试。

### 工具6：逆镀金词翻译表（业务沟通中常见的"镀金词" → 真实意思）

| 客户的话 | 真实意思 |
|---------|---------|
| 我们想要做一次成本-效益分析 | 我们要砍预算，不会真的考虑效益 |
| 给我们一个灵活性强的方案 | 我希望随时改主意 / 随时开人 |
| 我们追求质量 | 我不知道质量怎么定义 |
| 战略调整 | 出了问题但不想承认 |
| 优化人员结构 | 准备裁员 |
| 寻求战略合作伙伴 | 缺钱 |
| 暂时搁置 | 已死 |
| 我们已经反映给上级 | 我也没办法 |

## 💡 重要洞见与案例

### 洞见1：顾问的真正本事是"看见"，而不是"专业"

董事会议室里的大象——森林专家说像树、鲸学家说像鲸、蛇学家说像蛇。最后是清洁工掏出花生说："我也不知道是什么，但花生总能让大动物动起来"。

### 洞见2：客户用什么委婉语，就藏什么真问题

逆镀金法则的威力。当客户说"灵活性"，他真正想说的是"想开人就开人"。当HR说"成本-效益分析"，真意是"光分析成本不分析效益"。  
**应用**：每次见客户，把他用的"高级词"全部翻译成大白话写下来——这是诊断的金矿。

### 洞见3：清洁工视角 = 顾问的最强外援

第5章中的真实场景：解决程序员效率问题，跑去问清洁工"这公司缺什么"。清洁工说"他们从来不让我擦黑板"。这一句话引出了整个组织"工具用法变形"的核心问题。

### 案例：希德的链锁冰箱 → 后来的报警冰箱失败

希德把冰箱用自行车锁锁起来，效果好；后来换成"你好死胖子"的电子提示器，反而失败。原因：①声音太晚（已经看到食物了）；②让他想起被父亲嘲笑——触发抵御反应而非反思。
