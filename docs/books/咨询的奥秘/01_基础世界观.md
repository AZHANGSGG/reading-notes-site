# 第一部分：咨询的基础世界观

涵盖：第1章「咨询为什么这么难」、第2章「培养矛盾的思维框架」

---

## 🎯 核心主张

**做顾问的本质是接受三个反直觉的事实：客户不会真心承认自己有问题、解决一个问题只会让下一个问题升级、世界本质是矛盾的所以不能只用逻辑做事。能在这三件事上不崩溃，才是顾问的入场券。**

---

## 📌 关键框架：六组核心定律

### 1. 舍比咨询四定律（顾问与客户关系的底层物理）

| # | 定律 | 翻译成人话 |
|---|------|-----------|
| 第一 | 不管客户和你说什么，问题总会有 | 客户雇你时永远会说"我们这儿其实没啥问题"——这恰恰是问题最大的信号 |
| 第二 | 不管一开始看起来什么样，它永远是人的问题 | 技术问题、流程问题、合规问题，最终都能追溯到某个人的作为或不作为 |
| 第三 | 客户是按小时付费，不是按解决方案付费 | 真要按解决方案付费的话，问题早被自己解决了 |
| 第四 | 要是他们没聘用你，不要帮他们解决问题 | 不请自来的"帮助"=职业自杀 |

**推论族（每条都可单独用）：**

- **百分之十承诺定律**：永远不要承诺超过10%的改进——超过这个数客户就要被迫承认自己有大病，反而不会聘你
- **百分之十解决方案定律**：万一改进超过10%，确保没人注意到——把功劳全归给客户
- **马文定律**：不管客户在做什么，建议他们做点别的（因为埋头做的人正在重复"第一次就没成功的做法"）
- **功劳法则**：你要是在意功劳记在谁头上，就啥事也干不成
- **独行侠幻想**：如果客户不欣赏你，就假装他们被你的表现惊呆了——但记住这只是你的幻想

### 2. 树莓酱定律（影响力 vs 收入的物理约束）

> 铺得越广，摊得越薄。要名还是要利，自己挑吧。

刷盘子（一年9000美元）→ 顾问（30000）→ 培训师（50000）→ 演讲家（80000）→ 畅销作家（150000+）。受众越大、收入越高，但每个人能从你这儿拿到的"营养"就越少。这条定律和热力学第一定律一样硬。

### 3. 温伯格双胞胎定律（系统的惯性，咨询乐观情绪的解毒剂）

> 大部分时间，在世界上大多数地方，不管人们有多努力，都不会发生什么大事。

预测明天天气，"和今天一样"准确率66%。预测人的行为，"和昨天一样"准确率99%。这条定律是用来劝你**降低改变世界的不切实际的期待**——你接的咨询项目，大概率不会带来你想象中的剧变。

### 4. 鲁迪黄萝卜理论（解决问题的悲剧性）

> 一旦你干掉了头号问题，二号问题就升级了。

希尔曼超市13岁的温伯格搬走滞销黄萝卜——鲁迪问："那现在，哪个菜最卖不动呢？"  
**水龙头修好了，天线开始响。**

### 5. 困难三定律（什么样的人才能做下去）

| 定律 | 内容 |
|------|------|
| 困难定律 | 要是不能接受失败，做顾问就永远不会成功 |
| 更困难定律 | 一旦消灭了头号问题，你就让二号问题升级了 |
| 最困难定律 | **帮助自己比帮助别人更难** |

### 6. 矛盾思维三悖论（顾问的精神底色）

> - 不要理性，要合理。
> - 自以为无所不知的人最容易上当。
> - 生活太重要，所以不能太较真。

**Aaron Aardvark 案例**：工资程序卡死，所有"逻辑"调试都失败。温伯格凭空虚构一个员工"Aaron Aardvark"放在考勤卡最前面，跑通了。不是逻辑——是合理。

## 🔧 可落地的交付物

### 工具2：折中图（治疗"优化强迫症"的物理疗法）

**用法**：客户提出"最快/最便宜/最好"任何一个最高级形容词时，立刻在纸上画一张二维曲线图：

```
性能A
  ↑
  │ ╲
  │   ╲___
  │       ╲___
  │           ╲___
  └──────────────→ 性能B
```

然后问对方：「你愿意牺牲什么？」

每个谈客户的会议，桌上都应该出现这张图。

### 工具3：三层咨询自检表（接每个新单子前问自己）

```
[ ] 这个客户是不是真的承认自己有问题？（咨询第一定律）
    → 如果还在说"其实没啥大问题"，准备好被反复消耗时间

[ ] 我准备承诺多少改进？是不是超过了10%？（百分之十定律）
    → 超过就降到10%以下；超额完成的部分悄悄归功给客户

[ ] 这是不是被定义成了"技术问题"？背后真正的"人"是谁？（咨询第二定律）
    → 找到那个真正能拍板/改变行为的人，他才是项目能否成功的关键

[ ] 这是不是客户提出的"第三个问题"？（第三次魔咒）
    → 第一第二个问题往往是表象；客户磨合到提出第三个时才是真问题

[ ] 我现在去做这件事，是为了帮客户，还是为了让自己有事可做？
    → 如果是后者，可能违反了咨询第四定律
```

### 工具4：费舍基本定理 → 顾问反而是"未适应当前环境"的优势

> 你越适应现状，就越难适应变化。

## 💡 重要洞见与案例

### 洞见1：顾问的钱是"借口费"，不是"解决方案费"

经理花钱请顾问，本质上是**买一个可以向上司交代的借口**。"我们意识到有问题、努力解决了、还请了顾问"——只要这套话说出去，钱就花得值，不管问题解没解决。

### 洞见2：第三次魔咒——前两个问题都是诱饵

客户最初告诉你的问题，永远不是真问题。第一个是"礼仪性"的，第二个是"试探性"的，**到第三个才会出现真东西**。这也解释了为什么短期咨询合同（1-2个月）经常没效果——根本没活到第三次魔咒。

### 洞见3：温伯格双胞胎定律 → 顾问要降低自己的"野心"

「我们让你那家公司彻底改变文化」这种话，在双胞胎定律下基本是骗人的。99%的事不会变。顾问能做的，是在那1%能变的事上，把概率从5%推到15%。这就是巨大的价值了。

### 洞见4：独行侠幻想是顾问的精神保护机制

客户拿到方案后从来不会感谢你。他们会把功劳归给自己，甚至会忘记找过你。这件事会一次又一次伤害顾问。**温伯格的解药**：在心里造一个戏剧性的退场——"老师们摇着头说：那个蒙面女侠到底是谁呢？"——然后骑马进入夕阳。

这不是消极心理学，这是**保护职业续航能力的必要工具**。
