# 咨询的奥秘

> Gerald M. Weinberg

> **The Secrets of Consulting: A Guide to Giving and Getting Advice Successfully** by Gerald M. Weinberg (1986)  
> 中文版 2013 / 人民邮电出版社·图灵社区 / 劳佳 译

---

## 一句话总结

**做顾问的全部秘诀只有一句话：在他人请求时影响他们的艺术——而做好这件事的核心，是接受这个世界的反直觉物理学（解决一个问题会让下一个问题升级、付钱越多客户越爱你、你想留住的东西最容易失去），然后用最少的干预把卡住的系统轻轻一晃。**

## 全书定律全集（按主题归类）

### 一、咨询的基础物理学（4+8）

```
舍比咨询四定律：
  第一：不管客户和你说什么，问题总会有
  第二：不管一开始看起来什么样，它永远是人的问题
  第三：永远别忘了客户是按小时付费，不是按解决方案
  第四：要是他们没聘用你，不要帮他们解决问题

推论族：
  百分之十承诺定律：永远不要承诺超过10%的改进
  百分之十解决方案定律：万一改进超过10%，确保没人注意到
  马文定律：不管客户在做什么，建议他们做点别的
  功劳法则：在意功劳归属就啥事干不成
  独行侠幻想：客户不欣赏你，就假装他们被惊呆了
  树莓酱定律：铺得越广，摊得越薄；要名要利自己挑
  温伯格双胞胎定律：大部分时间，不管多努力都不会发生大事
  鲁迪黄萝卜理论：消灭头号问题，二号就升级了
```

### 二、顾问的精神底色（3+3）

```
困难三定律：
  困难定律：不能接受失败，做顾问就永远不会成功
  更困难定律：消灭头号问题就让二号升级
  最困难定律：帮助自己比帮助别人更难

矛盾思维三悖论：
  不要理性，要合理
  自以为无所不知的人最容易上当
  生活太重要，所以不能太较真
```

### 三、看见与诊断的工具（约15条）

```
折中疗法：不付出就什么也得不到 / 提升一方面就要牺牲另一方面
费舍基本定理：你越适应现状，就越难适应变化
第三次魔咒：顾问在解决你的第三个问题时最有效
橙汁测试：我们能做。这是所需的费用

马文医学六大秘密：
  头号：90%的病都自愈 → 温柔对待能自愈的系统
  第二：反复治疗自愈系统会让它失去自愈能力
  第三：每个处方=药品+正确使用方法
  第四：已经做过的事没解决问题，就告诉他们做点别的
  第五：让他们付足够多的钱，他们才会照你说的做
  第六：来得早不如来得巧

伯登/镀金/逆镀金：
  伯登法则：要是你不能改掉缺点，就把它变成特点
  镀金法则：要是没法当成特点宣扬，那就冒充一下
  逆镀金法则：所有镀金的东西都得改正

锤子定律：圣诞收到锤子的孩子会发现所有东西都需要敲打
白面包警告：用同样的做法，就会得到同样的面包
伯丁追溯原理：事情是一步步变成现在的样子的
斯巴克斯解决问题定律：越接近找出造成问题的人，解决机会越小
"为什么"诅咒：永远都不缺理由

研究指南：
  保持简单，不要太详细——你是顾问，不是律师
  研究是为了理解，不是为了批评
  在现状中寻找你喜欢的东西并加以赞美

标签法则：我们买的是标签，不是商品；事物的名称不是事物本身
三个手指法则：你用一根手指指别人时，看看另外三根指哪里
五分钟法则：客户永远在头五分钟讲出怎么解决自己的问题
平均定律：高效问题解决者有很多问题，但很少有特别重大的一个
温伯格离奇定律：有时候觉得离奇只是因为眼界不够
三的法则：想不出三处可能问题，思维肯定哪里出了问题
对不一致的洞察力：语言和音乐不搭配的时候就指出了一个缺漏
布朗的天才遗产：语言常有用，但听听音乐总有好处
主管线格言：你不知道的不会伤到你，但你记不得的肯定会
薯片原理：了解你的受众，就很容易设置提示物
泰坦尼克效应：以为灾难不可能发生往往导致不可想象的灾难
```

### 四、影响与改变（约15条）

```
晃动定律 / 干预第一定律：少即是多
逆温伯格定律：有些时候，会发生重大变化，特别是人们没努力改变的时候
变化第一定律 / 普雷斯科特腌黄瓜原则：黄瓜被卤水腌的多，卤水被黄瓜染的少
流浪者法则：努力留在家会让你变成流浪者
留守者法则：努力去旅行会让你变成留守者
罗默法则：失去一个东西的最好办法就是努力留住它
快餐谬误：没区别+没区别+没区别 = 很大的区别
福特基本反馈公式：从河里取水的人必须把等量水还回上游
温伯格测试：你愿意把自己的生命托付给这个系统吗？

"新"的定律 / 潘多拉的疹子：
  新玩意儿从来就不好使
  新玩意儿从来不好使，但我们总是希望这一次会有所不同

发牌员的选择：信任每个人，但要自己来切牌
埃德塞法令：必须搞点新东西，那么只要一个，不要两个
大众汽车真理：如果你不能拒绝它，就去化解它

时间炸弹：
  时间会击破所有弱点
  浪费时间的最可靠的方法就是扔掉谨慎

隆达三启示：
  危机其实只是幻觉的破灭
  当变化无可避免时，我们会尽力保持最看重的东西
  建立幻觉来防止变化的时候，改变就更可能发生

水牛缰绳：水牛只去它想去的地方
```

### 五、营销十定律（含琳恩生存法则）

```
1. 顾问只有两种状态：闲（I）或忙（B）
2. 最好的找客户的方式是有客户
3. 每周至少花一天时间来曝光自己
4. 客户对你总是比你对他们更重要
5. 永远不要让一个客户占总业务的1/4以上
6. 最好的营销工具是满意的客户
7. 把最好的想法送人
8. 自己加了个蛋之后味道更好（邓肯·海因斯的差异）
9. 至少留出1/4的时间什么都不做
10. 为质量营销，而不是为数量（夏洛特定律）

琳恩生存法则：为了能对自己说"是"，你要能对任何客户说"不"
```

### 六、定价十定律（含最小遗憾原则）

```
1. 定价有许多功能，金钱交换只是其中一个
2. 付钱越多对你的爱越多；付钱越少对你的尊重越少
3. 钱一般是价格中最小的一部分
4. 定价不是零和游戏
5. 如果你需要这份钱，就别接这个活
6. 如果他们对你的工作不满意，就不要拿他们的钱
7. 金钱不只是价格
8. 价格不是物品，是谈判达成的关系
9. 设定的价格应该让你不论谈没谈成都不后悔（最小遗憾原则）
10. 所有价格最终都是基于感觉的
```

### 七、信任十定律（含黄金法则）

```
1. 除了你自己，没人在乎你让别人失望的理由
2. 信任需要多年才能赢得，但只需一瞬间就可失去
3. 人们不再信任你的时候不会告诉你
4. 赢得信任的技巧就是避免所有的花招
5. 人是绝不会撒谎的——在他自己眼中
6. 永远信任你的客户，不过要自己切牌
7. 就算客户要求，也永远不要不诚实
8. 永远不要做任何承诺
9. 永远信守诺言
10. 把它写下来，但还是要靠信任

黄金法则（第十一定律）：留个空位给所有定律没涵盖的、终极的信任原则
```

### 八、农场园艺六心法（最高境界）

```
1. 永远不要用便宜的种子
2. 精心准备土壤是所有园艺的秘诀
3. 时机至关重要
4. 长得最牢的是自己长根的种子
5. 浇水太多会让它变弱而不是变强
6. 尽管你尽了最大的努力，有些植物还是会死掉
```

## 阅读和应用建议

1. **第一遍精读**：按1-2-3-4顺序读，建立完整地图
2. **第二遍带问题读**：当遇到具体业务问题时，回到对应分册查找定律
3. **建立"季度复盘"**：每季度选1-2条定律，对照过去90天的实际工作做反思
4. **建立"年度园艺六心法"自检**：每年初按6条心法重新规划当年的"种植"

## 一些不在主线但值得标注的金句

> 顾问是应人们的要求去影响他们的艺术。

> 因为人们需要某种变化或是担心发生某种变化，所以他们会去做这样那样的咨询。

> 真相可能会让人受伤，但幻觉会让人伤得更深。

> 没有人推的时候，要抗拒是很难的。

> 你也许愿意成为拿着高薪的替罪羊，但你至少应该头脑清醒地做出选择。

> 顾问最常见的职业病就是不请自来地提供"帮助"。

> 我的很多客户都让我充当"镜子"——用来看清他们自己。

> 大多数人都和斯巴克斯一样没耐心去了解历史。这也就是为什么了解历史是一个让顾问看到其他人忽视之处的好办法。

> 我桌子上一直放着的一首小诗：  
> 若需一束光，请做一颗星，让它传得广，让它传不停；  
> 但你若想要，让太阳改变，还需从头起，一步一脚印。
