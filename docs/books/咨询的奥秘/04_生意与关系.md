# 第四部分：生意与关系——营销/定价/信任/园艺心法

涵盖：第11章「推广你的服务」（10条营销定律 + 琳恩生存法则）、第12章「把价签贴在脑门上」（10条定价定律）、第13章「怎样赢得信任」（10条信任定律 + 黄金法则）、第14章「让别人采纳你的建议」（农场园艺6心法）

---

## 🎯 核心主张

**做顾问这门生意的本质规律有三条：营销最有效的时刻是你最忙的时刻、定价反映的是客户对你的"爱"和"尊重"而不是工作量、信任只能用绝对的诚实和不耍花招获得。所有这些定律最终都在说同一件事——你所有的"商业行为"，都在向客户传递关于你的"自我价值评估"。**

## 📌 关键框架

### 1. 十大营销定律（顾问的生意物理学）

| # | 定律 | 一句话精髓 |
|---|------|-----------|
| 1 | 顾问只有两种状态：闲（I）或忙（B） | "刚好的业务量"不存在；接受这点，别期待中间状态 |
| 2 | 最好的找客户的方式是有客户 | 富者愈富。营销的最佳时机正是你最忙的时候 |
| 3 | 每周至少花一天曝光自己 | 哪怕已经状态B，也不能省 |
| 4 | 客户对你总是比你对他们更重要 | 你以为铁打的合同，他们一个电话就砍掉 |
| 5 | 永远不要让一个客户占总业务的1/4以上 | 否则失去对自己说"是"的能力 |
| 6 | 最好的营销工具是满意的客户 | 90%新业务来自老客户的间接介绍 |
| 7 | 把最好的想法送人 | 紧抓不放 = 创新枯竭 = 顾问生涯倒计时 |
| 8 | 自己加了个蛋之后味道更好（邓肯·海因斯的差异） | 让客户参与方案的小部分，他们就会爱上整个方案 |
| 9 | 至少留出1/4的时间什么都不做 | 没有空闲就接不住突发机会，也没法为新想法充电 |
| 10 | 为质量营销，而不是为数量（夏洛特定律） | 半瓶欢乐牌洗洁精事件——比所有其他9条加起来还重要 |

**琳恩生存法则**（凌驾于以上定律）：  
> 作为顾问，**为了能对自己说"是"，你要能对任何客户说"不"**。

### 2. 十大定价定律（价格的物理学）

| # | 定律 | 含义 |
|---|------|------|
| 1 | 定价有许多功能，金钱交换只是其中一个 | 一美元卖不出的唇膏，五美元卖断货 |
| 2 | 付钱越多对你的爱越多；付钱越少对你的尊重越少 | 同样工作，免费做累10倍 |
| 3 | 钱一般是价格中最小的一部分 | 心理成本+人力+时间+麻烦 都是价格 |
| 4 | 定价不是零和游戏 | 我的收益未必等于你的损失（旅行机会、新听众、双向学习） |
| 5 | 如果你需要这份钱，就别接这个活 | 焦虑感会被客户感知，反而拿不到 |
| 6 | 如果他们对你的工作不满意，就不要拿他们的钱 | 满意保证制 |
| 7 | 金钱不只是价格 | 不可退还预订金、阶段付款——都是工具 |
| 8 | 价格不是一件物品，它是谈判达成的关系 | 找一个比你更善于谈价的人代谈 |
| 9 | 设定的价格应该让你不论谈没谈成都不后悔（最小遗憾原则） | 决策标准 |
| 10 | 所有价格最终都是基于感觉的 | 你的+他们的 |

**最重要的一条：你必须知道自己值多少钱**——这是所有定价问题的根源。

### 3. 十大信任定律 + 黄金法则

| # | 定律 | 含义 |
|---|------|------|
| 1 | 除了你自己，没人在乎你让别人失望的理由 | 别再解释 |
| 2 | 信任需要多年才能赢得，但只需一瞬间就可失去 | 没有"再次证明自己"的机会 |
| 3 | 人们不再信任你的时候不会告诉你 | 他们会沉默地不再请你 |
| 4 | 赢得信任的技巧就是避免所有的花招 | 唯一的"绝招"=坦诚直接 |
| 5 | 人是绝不会撒谎的——在他自己眼中 | 客户不是骗子，只是"简化/带过/省略" |
| 6 | 永远信任你的客户，不过要自己切牌 | 信人格，不信能力 |
| 7 | 就算客户要求，也永远不要不诚实 | 帮一次小忙=永远丢信任 |
| 8 | 永远不要做任何承诺 | 即使非正式的也不行 |
| 9 | 永远信守诺言 | 已经做过的事=暗示未来许可 |
| 10 | 把它写下来，但还是要靠信任 | 没合同的信任 > 没信任的合同 |

**黄金法则**（第十一诫，温伯格也想不起来内容）：  
> 留个空位给那条所有定律没涵盖的、终极的信任法则。

### 4. 农场园艺6心法（让别人采纳你建议的最高境界）

约翰是温伯格的园艺顾问，10年间从不直接给答案，只指出原理。这6条是**所有咨询的元法则**：

```
1. 永远不要用便宜的种子
   → 找点子的成本，相比把点子培养出来的成本，可以忽略
   → 在花大价钱培育之前，要尽一切可能找到最好的点子

2. 精心准备土壤是所有园艺的秘诀
   → 看不见的工作（土壤）决定看得见的成果（植物）
   → 在种下点子之前所做的准备工作，决定点子有效与否

3. 时机至关重要
   → 顾问很容易一想出点子就四处宣扬
   → 而不是等待适合它发芽的那一刻

4. 长得最牢的是自己长根的种子
   → 把土壤压实让植物抓住，根反而不强壮
   → 给点子的"保护"越少，根扎得越牢

5. 浇水太多会让它变弱而不是变强
   → 过度宣传 / 过度施肥 / 过度投入 = 想法变虚弱
   → 想法和植物一样，都需要一定程度的挣扎才能茁壮成长

6. 尽管你尽了最大的努力，有些植物还是会死掉
   → 不是每个客户都能救活
   → 学会承受失败而不往心里去
```

## 🔧 可落地的交付物

### 工具1：营销日历——一周必做的固定动作

```
周一: 公众号/视频号产出（曝光+1天）
周二: 客户A工作日
周三: 客户B工作日 + 1次老客户回访
周四: 客户C工作日
周五: 业务发展（潜在客户咖啡/演讲/写作）
周末: 营销第九定律——什么都不做
```

每周必须有：1天曝光、4天客户工作、1天回访老客户、留1.5天空白。

### 工具3：客户健康度月度复盘表

每月最后一天，按这个表评估所有客户：

| 客户 | 占总业务比例 | 这个客户让我能否说"不"？ | 满意度（1-5） | 是否值得续约？ |
|------|-------------|------------------------|--------------|---------------|
| A | 30%（超标！） | 否——已经不敢说不 | 4 | 减仓 |
| B | 15% | 是 | 5 | 续 |
| ... |

**红线**：单个客户超过25%就必须降负重。

### 工具4：拒绝不诚实请求的标准话术（信任第七定律）

当客户说"帮我个小忙，不用太严格"——

 我宁可这次让你不开心、退回部分费用让你找别人做，
 也不愿意以后我们的合作建立在'看情况而定'之上。

 所以我提议另一个方案：[替代方案]"
```

### 工具5：信任崩塌后的诊断步骤

当某个老客户突然冷淡 / 减少咨询 / 不接电话——按这个清单复盘：

```
[ ] 我最近一次让他失望是什么时候？
[ ] 当时我有没有花时间解释？（如果有，反而错了——信任第一定律）
[ ] 我有没有做过任何"花招"？（迂回拜访、暗中调查、未经允许的联系）
[ ] 我是不是答应过什么没做到？（哪怕是非正式的）
[ ] 我有没有不知不觉变成了"自己加蛋"的反面——把客户排除在外了？
[ ] 我有没有让他觉得我"不需要他"了？（违反第四定律：客户对我比我对他更重要）
[ ] 我有没有对他说过任何"小白谎"？

对应的修复动作：
- 如果是"花招"问题：诚恳直接道歉，不解释原因
- 如果是承诺问题：主动提出补救（重做+折扣）
- 如果是排除问题：邀请他参与一次"加蛋"的咨询过程
- 其他情况：直接问"我做了什么让你不舒服的事吗"
```

### 工具6：园艺式咨询自检——给老客户提建议前必问

每次想给老客户提"我有个新建议"之前，过这6条：

```
[ ] 这是不是"便宜种子"？我用最好的精力找过更好的方案吗？
[ ] 客户的"土壤"准备好了吗？还是只是我急？
[ ] 时机对不对？现在提是不是太早/太晚？
[ ] 我是不是想"压实"这个想法？能不能让客户自己长根？
[ ] 我是不是在过度浇水（推太用力）？
[ ] 万一这个建议失败了，我能接受吗？
```

## 💡 重要洞见与案例

### 洞见5：最小遗憾原则是定价的精神纪律

不要纠结"市场价"或"客户能承受多少"。问自己一个问题：被接受我会后悔吗？被拒绝我会后悔吗？  
**找一个两种结果都不后悔的价格点。**

### 洞见6：人不会对自己撒谎——所以不要指责

客户给错信息，他们说"我以为简化更好"或"我觉得不重要"——他们真的是这么认为的。指责他们撒谎=信任关系结束。**这是顾问最容易犯的最致命错误之一**。

### 洞见7：信任有两个独立的维度——人格 vs 能力

福勒不是骗子（人格OK），但他不可靠（能力差）。客户既需要诚实，也需要可靠。**要分开评估对方在两个维度上的得分**——只在自己有充分把握的维度上信任。

### 洞见8：合同不能代替信任——但要把它写下来

福勒老师教的"把它写下来 × 3"是对的。但**没合同的信任 > 没信任的合同**。书面的作用是预防误解，不是替代信任。

### 洞见9：约翰的园艺哲学——10年只指原理不给答案

温伯格用10年才意识到约翰是他的"顾问"——这恰恰证明约翰是完美的顾问。**最高级的咨询，是让客户10年后才意识到你帮过他**。

### 案例：夏洛特和欢乐牌洗洁精

夏洛特是律师，做到状态B。一个离婚案丈夫给的财产分割单，写"欢乐牌洗洁精中瓶装，三分之一满，归丈夫"——夏洛特看着这行字，决定从此不接离婚案。

**这就是夏洛特定律的诞生**：你不是赚钱机器，你最终要问"我真的想做这玩意吗"。
