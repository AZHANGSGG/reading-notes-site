# 第二部分·商业应用:破、立、见的三种打法

> 第一部分讲了什么是第一性原理。这一部分讲怎么用。
> 三种打法:**破界创新**(打破别人的)、**组织刷新**(打破自己的)、**第一创新**(立全新的)。
> 案例载体:计算机产业链 → 微软纳德拉 → 马斯克。

---

## 第三章·破界创新:打破基石,边界外延

### 🎯 核心主张

任何系统的边界都被它的第一性原理决定;创新不是在系统**之内**做更多事,而是从系统**之外**打破第一性原理——这是从内向外的创新,不是从外向内的优化。

### 📌 关键框架/方法论

**1. 破界创新三部曲**

```
第一步「破」隐含假设  ← 最难的一步,也是真正的创新发生处
       ↓ (用哲学思维跳出系统)
第二步「立」基石假设  ← 重新定义系统的基石,必须比原假设层次更深
       ↓ (基石必须来自更基础的学科)
第三步「见」全新系统  ← 顺理成章,水到渠成
```

**2. 隐含假设的三个特征**

- **隐**:你已经把它当真理,根本没意识到这是个假设
- **群体共识性**:周围人都信→社会都信→变成"真理",这恰恰是它最危险的地方
- **基石性**:推翻它,整个上层结构会一起塌(地基的力学位置)

> 「最常见的隐含假设就是群体信念,你身处群体信念之内,却难以察觉」

**3. 破界创新 vs 普通创新**

| 维度 | 普通创新(组合/单点) | 破界创新 |
|---|---|---|
| 武功比喻 | 外家功夫,招式繁多 | 内家功夫,一击制胜 |
| 创新方向 | 由外而内 | 由内而外 |
| 改变对象 | 外部产品/流程 | 内部认知边界 |
| 创业者类型 | 解题家(解决既有问题) | 理论家(让既有问题失去意义) |

**「真正的破界创新不是解决原问题,是让原问题变得无关紧要」**——这句话是这一章的精髓。

### 🔧 可落地的交付物

**交付物1·"识别我所在系统的隐含假设"五问法**

把这套问题打印出来贴在墙上:

1. 我所在行业,所有人都默认成立的"常识"有哪些?
2. 这些常识背后的隐含假设是什么?(注意:常识是结论,假设是支撑)
3. 这些假设是用什么逻辑被证明的?是归纳法吗?
4. 如果环境改变(地理、时代、技术、客户群),这些假设还成立吗?
5. 如果其中一个假设被打破,会产生什么样的新可能?

**交付物2·破界创新三部曲实操模板**

```
我要做的事:_______________

第一步「破」:
  当前行业的隐含假设是______________(必须是大家都不说但默认的)
  我用哪种"哲学思维"识别出它的?

第二步「立」:
  我重立的基石假设是______________
  这个新假设比原假设深在哪里?(基础学科的支撑是什么?)
  ⚠️ 检查:如果新假设和原假设在同一层级,那只是观点之争,不是破界

第三步「见」:
  在新基石上,系统的具体形态是______________
  (这一步如果做不到顺理成章,说明前两步还没到位)
```

**交付物3·"理论家 vs 解题家"自我定位**

> 解题家:解决既有问题(优化电池续航、压缩屏幕尺寸)
> 理论家:让既有问题失去意义(乔布斯把屏幕去掉、福特造汽车不造马车)

诚实自问:**你的产品,是在解决既有问题,还是在让既有问题失去意义?**

### 💡 重要洞见与案例

**1. 计算机行业30年破界创新链**——这一章最值钱的案例,直接当故事讲

```
雷明顿兰德(优尼瓦克,1951)
  隐含假设:计算机只能用于科学研究
  → 销售逻辑:不需要派销售员,因为企业人不懂计算机
  → 1950年市场调研:到2000年全世界总销售1000台
  现实:1984年就突破100万台

       ↓ IBM破界

IBM(1952,商用机)
  新基石假设:企业市场是计算机的潜力市场
  动作:把100万美元的科研机降到10万美元
  结果:垄断大型商用机市场

       ↓ DEC破界

DEC(1959,小型机)
  新基石假设:小公司也需要计算机
  动作:1.2万美元(IBM的1/10)
  结果:1972年彻底控制小型商用机市场

       ↓ 苹果破界

苹果(1976,个人计算机)
  新基石假设:每个家庭都可以拥有一台计算机
  动作:1295美元(DEC的1/10)
  结果:开启个人计算机时代

       ↓ 但DEC的反讽

DEC创始人Olsen,1977年世界未来学大会上说:
"我觉得没有理由每个家庭都有一台计算机"
DEC的隐含假设依然是"计算机只能用于商业市场",最终错失PC时代
```

**这个案例链的真正洞察**:

> **「屠龙的少年最终会变成恶龙」**——你赢的那个隐含假设,几年后就是你输的那个隐含假设

每一代赢家都是因为打破了上一代的隐含假设而崛起,但他们立的新假设,又会变成下一代要打破的隐含假设。

**2. 乔布斯的"重新定义"系列**

| 产品 | 当时的群体认知 | 乔布斯的新基石假设 |
|---|---|---|
| iPod Shuffle | 播放器必须有屏幕来选歌 | 用户根本不需要选,随机就好 |
| 所有苹果产品 | 电子产品必须有开关键 | 不操作就休眠,触摸就唤醒,要开关干嘛 |
| iPhone | 手机必须有固定键盘 | 触摸屏可以取代键盘 |

> **乔布斯的核心特征是"因为想到,所以看到",而我们大多数人是"因为看到,所以想到"**

**3. "更快的马车"金句——破界创新的终极注解**

亨利·福特:**「如果你问消费者需要什么,他会回答说需要速度更快的马车」**

李善友的解读链:
- 福特没解决马车问题,他让马车问题变得无关紧要
- 乔布斯没解决功能机问题,他让功能机问题变得无关紧要
- 马斯克没解决传统汽车问题,他让传统汽车问题变得无关紧要
- 张小龙没解决PC即时通讯问题,他让PC即时通讯问题变得无关紧要
- 张一鸣没解决门户新闻问题,他让门户新闻问题变得无关紧要

→ **这个公式可以套到任何"颠覆者"身上,也可以反过来套你自己**

### 📖 值得记录的金句

- 「破的是系统得以形成的第一性原理,立的是新的第一性原理」
- 「最常见的隐含假设就是群体信念」
- 「破界创新不是在内容上做功,而是在结构上做功,是在看不到的地方做功」
- 「身处系统之中,我们用常规的思维方式永远看不到系统本身的隐含假设——只有跳出系统才能看到系统的边界,就像鱼跳出水面才能看到自己生活在水中」
- 「破界创新并没有解决原有系统的极限问题,但是让原有系统的极限点问题变得无关紧要」

## 第四章·组织刷新:使命—战略的破界创新(纳德拉案例)

### 🎯 核心主张

组织变革不是从战略开始,是从使命开始;使命是战略的隐含假设;刷新使命→刷新文化→刷新战略,这是企业重新激活的唯一可行路径。

### 📌 关键框架/方法论

**1. 组织刷新三步法**

```
第一步:刷新使命  ← 找到组织的"灵魂"——我们为什么存在?
       ↓
第二步:刷新文化  ← 改变思维结构,不是改变内容
       ↓
第三步:刷新战略  ← 自然涌现,不需要单独"设计"
```

**关键反直觉**:大多数人变革的顺序是反过来的(战略→文化→使命),所以失败。

**2. 微软的失速点分析**

| 维度 | 鲍尔默时代(2000-2014) | 真相 |
|---|---|---|
| 表面成绩 | 营收增长4倍(778亿)、利润10倍(267亿) | 数字漂亮 |
| 真实变化 | 市值缩水近一半 | 第二曲线全面失守 |
| 错过的浪潮 | 互联网搜索、社交、移动、云计算 | 系统性错过 |
| 根本原因 | 被"PC+Windows+Office"的隐含假设禁锢 | 不是被对手打败,是被自己禁锢 |

> 鲍尔默的标志性话:2013年还在说「在微软没有什么比Windows更重要」

**3. 固化型思维 vs 成长型思维(纳德拉的文化刷新核心工具)**

来自卡罗尔·德韦克《终身成长》。

| 固化型思维 | 成长型思维 |
|---|---|
| 能力天生不变 | 能力可以通过努力提高 |
| 看结果(我做错了什么) | 看过程(我学到了什么) |
| 逃避风险却期待创新 | 拥抱失败,从失败学习 |
| 必须证明自己最聪明 | 不怕承认自己不知道 |
| 部门之间打架,内卷 | 部门之间合作,做朋友 |

> 「固化型思维的人天天想着逐渐下滑的PC市场。成长型思维会从守势转变为攻势,目标是要赢得价值数十亿美元的联网设备」——纳德拉

### 🔧 可落地的交付物

**交付物1·组织变革的"自我灵魂拷问"清单**

要做组织刷新,先问这5个问题:

1. 我们的公司是做什么的?
2. 我们为什么而存在?
3. 如果我们公司明天消失,会对这个世界有什么影响吗?
4. 我们的使命已经实现了吗?
5. 创始人当年提出原使命时,**他真正想做的事是什么?**(纳德拉的关键追问——盖茨真正的初心是"赋能他人")

→ **「赋能」不是从产品归纳出来的,是从盖茨初心演绎出来的**

**交付物2·"刷新文化"的三个抓手**

不要写文化口号,做这三件事:

1. **每天检查每一个业务决定是否帮助公司转向成长型思维**(CEO级别的元决策)
2. **公开做一件颠覆既有文化的事**(纳德拉公开用iPhone演示Outlook,这一动作胜过100次内部宣讲)
3. **系统性放权一件原本紧抓不放的核心利益**(微软放弃必应在雅虎的独占,放弃Windows手机授权费,放弃Office绑定Windows)

**交付物3·"使命驱动战略"的反向核查**

老板们常说"我们的战略是X"。反向核查:
- 你的战略X背后,隐含的使命是什么?
- 这个使命,你能用一句让员工感动到落泪的话说出来吗?
- 如果不能,大概率你没有真正的使命,你只有一个商业模式

### 💡 重要洞见与案例

**1. 纳德拉刷新使命的精彩动作——不是创造,是"重新发现"**

《刷新》英文版副标题:"Rediscovering Microsoft's Soul"(重新发现微软的灵魂)

新使命:**「赋能全球每一人、每一个组织,帮助他们成就不凡」**

→ **真正的使命不是新创的,是被遮蔽的。变革的工作是去除遮蔽,不是发明**。

**2. 纳德拉读《终身成长》的故事——管理学最重要的"无心插柳"**

纳德拉有个患脑部疾病的孩子,他太太给他推荐《终身成长》本来是为了孩子教育。但纳德拉读完发现:**这本书才是微软文化病灶的诊断书**。

→ 跨域学习的力量:管理书没解决的问题,儿童教育书解决了

**3. 纳德拉的"去Windows化"决策——壮士断腕的范式**

```
2014.2  纳德拉接任,微软市值3017亿
2015.9  公开用iPhone演示Outlook(文化信号)
2015    把Windows、Xbox、Surface打包到"More Personal Computer"业务
2016+   全面推Office到iOS/Android(放弃Windows绑定)
2018.3  Windows部门拆分并入新部门——「Windows」这个词从微软内部消失
2018.11 微软市值8512亿,超越苹果重回第一
2020.1  市值1.21万亿
```

> **「Windows既是光环也是枷锁,卸下它才能走得更远」**——纳德拉

**4. 一个具体打动人的细节**

鲍尔默时代,如果有高管在公开场合用Windows Phone之外的手机,**会被斥责**。
纳德拉上任后,亲自在发布会上用iPhone演示Outlook。

→ 这一个动作,胜过100份"开放心态"的内部备忘录。

### 📖 值得记录的金句

- 「文化把战略当早餐吃」——德鲁克
- 「最终任何一家企业的成功或失败的原因,都是其现行文化的滞后反应」——纳德拉
- 「卓有成效的战略变革,不是从内容着手,而是从结构着手」
- 「Windows既是光环也是枷锁,卸下它才能走得更远」——纳德拉
- 「微软并没有被任何竞争对手打败,而是被它自己内在的隐含假设禁锢」
- 「创始人的认知边界是一家企业发展的真正边界」

## 第五章·第一创新:基于第一性原理进行创新(芒格 + 马斯克)

### 🎯 核心主张

破界创新是"破"——打掉既有系统;第一创新是"立"——直接从第一性原理出发,演绎出全新系统。**第一创新更适合从零开始的创业者**。

### 📌 关键框架/方法论

**1. 破界创新 vs 第一创新**

| 维度 | 破界创新 | 第一创新 |
|---|---|---|
| 关键词 | 破 | 立 |
| 路径 | 具象→抽象→新具象 | 抽象→具象 |
| 难点 | 第一步识别隐含假设 | 找到正确的第一性原理 |
| 适用人 | 既有产业的颠覆者 | 起步阶段的创业者 |

**2. 系统层级关系——这是第一创新的入口**

```
自然科学 / 社会科学(母系统)
    ↓ 提供第一性原理
商业系统(子系统)
    ↓ 推导
具体业务模型
```

→ 「在商业系统中是找不到商业系统的第一性原理的,只能在更大的社会科学和自然科学中才能找到」

李善友的具体推导:
- 物理学的还原论 → 推出「组合创新」
- 生物学的进化论 → 推出「分形创新」  
- 哲学的本体论 → 推出「破界创新」

**3. 芒格的多元思维模型——避免"锤子综合征"**

> 「如果你只有一把锤子,你满世界看到的都是钉子」——芒格

| 概念 | 内涵 |
|---|---|
| 锤子综合征 | 用单一思维模型看世界,导致认知扭曲 |
| 多元思维模型 | 来自不同基础学科的思维框架的组合 |
| 数量 | 100个差不多通识,核心10个不到 |
| 来源 | 各学科真正的"大道理"——通常在大学一二年级课程里 |
| 训练方法 | 刻意练习,把模型植入潜意识 |

**4. 张首晟的"信封背后"思想实验**

末日方舟,只能带8条人类知识精华:
1. 哲学:原子论
2. 数学:欧氏几何
3. 生物学:自然选择
4. 经济学:看不见的手
5. 政治学:人人生而平等
6. 物理学:E=mc²
7. 物理学:熵增定律
8. 物理学:测不准原理

**5. 马斯克的还原论思维方法(操作版)**

```
Step 1. 拆解到原子级要素(物理学还原论)
        例:火箭→铝合金、钛、铜、碳纤维
Step 2. 计算原子级成本占比
        例:火箭原料只占总成本2%
Step 3. 发现真正的成本驱动因素是"组合方式"
        例:火箭贵是因为一次性使用
Step 4. 提出反共识假设
        例:火箭可以重复使用
Step 5. 用新组合方式重建系统
        例:猎鹰9号回收
```

### 🔧 可落地的交付物

**交付物1·"还原论拆解"五步法**(直接套马斯克的方法)

任何看似不可能/成本极高的目标:

```
目标:_______________

Step 1. 拆解到原子级:这件事的最基础物质/要素是什么?
Step 2. 算原子级成本:把所有原子要素加起来,真实成本是多少?
Step 3. 找成本黑洞:总成本 - 原子成本 = ? 这个差是哪来的?
Step 4. 找隐含假设:为什么这个差是合理的?背后是什么群体共识?
Step 5. 提出反共识:如果某个假设不成立,这个差会怎么变?
```

「在新加坡注册一家公司,真实必要成本是多少?」
- 原子级要素:政府注册费 + Bizfile费 + 公司秘书最低成本 + 注册地址
- 算下来真实必要成本可能不超过SGD 1500
- 市场上动辄几千甚至上万——这个差是哪来的?
- 隐含假设:客户希望省心、希望"跟着大家选"、希望有顾问背书
- 反共识:如果客户不需要这些"心理服务",成本可以做到多低?
- → 这就是产品差异化的真正源头

**交付物3·"厨子还是厨神"自检题**

蒂姆·厄本的比喻太好用:

| 厨子 | 厨神 |
|---|---|
| 严格按菜谱做菜 | 不按菜谱做菜 |
| 受既有规则束缚 | 致力于创新 |
| 学习方式:获取新信息 | 学习方式:提升思维模型 |

诚实自问:

### 💡 重要洞见与案例

**1. 马斯克拆解电动汽车电池**

```
电池总成本(2008): 600美元/kWh
电池原材料成本(从伦敦金属交易所买): 82美元/kWh
原料占比: 13.7%

真正的成本不在原料,在原料的组合方式
↓
和松下合作,采用18650钴酸锂电池管理程序

电池价格变化:
2008: 600美元/kWh
2014: 350
2016: 190
有了Tesla超级电池工厂后: 100美元/kWh(原料成本仅80)
```

**2. 马斯克拆解超级隧道——"四管齐下"**

美国地铁建造成本:6.25亿美元/公里
马斯克的目标:十倍好,即6250万美元/公里以内

```
拆解1:隧道直径减半 → 挖掘面积变为1/4
拆解2:边挖边加固墙壁 → 效率提升1倍
拆解3:挖掘机功率2倍 → 成本再减半
拆解4:挖出的废土做成砖块卖 → 创造收入冲抵成本

结果:
计划成本(1.83km的洛杉矶隧道):11亿美元
实际成本:1000万美元
百倍好,远超十倍好的目标
```

**3. 马斯克拆解火箭——最反共识的一击**

发现:
- 火箭原材料只占总成本2%
- **真正的成本黑洞:火箭一次性使用**
- 在马斯克之前,所有国家、所有公司都默认火箭=一次性
- 马斯克提出可重复使用——这就是站在系统之外看问题

结果:Space X火箭成本控制在同级别1/5,持续下降中。

**4. 马斯克的火星移民拆解——2.5万倍好**

小布什时代:一个人去火星100亿美元
马斯克目标:单人单程20万美元(2.5万倍好)

拆解:
- 一次送1人 → 一次送100-200人(1000艘飞船)
- 火箭单程使用 → 飞船重复使用12-15次,火箭重复100次
- 全程带燃料 → 太空轨道建燃料补给站
- 全程带燃料 → 在火星制造返程燃料(选甲烷,因为火星上易制造)
- 现有燃料 → 高效便携新燃料

**5. 张一鸣 vs 李志刚的对话——做的是"一件事"**

李志刚:「你们做多元化」
张一鸣:「你觉得我们多元吗?我觉得我们挺专注的,**它们是一个东西**」

→ 这就是第一创新的极致:**从外部看是多元化,从内部看是同一件事的不同表达**。
